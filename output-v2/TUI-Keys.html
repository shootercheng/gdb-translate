25.2 TUI 按键绑定 ¶

TUI 在 readline 键映射中安装了几个按键绑定（请参阅 命令行编辑）。 以下按键绑定同时适用于 TUI 模式和 GDB 标准模式。

C-x C-a ¶
C-x a
C-x A

进入或退出 TUI 模式。退出 TUI 模式时， curses 窗口管理停止， GDB 并使用其标准模式，直接在终端上进行操作。重新进入 TUI 模式时，控制权交还给 curses 窗口。 然后刷新屏幕。

此按键绑定使用可绑定的 Readline 函数 tui-switch-mode.

C-x 1 ¶

使用仅包含一个窗口的 TUI 布局。该布局将是 ‘源代码’ 或 ‘汇编代码’。当 TUI 模式未激活时，它将切换到 TUI 模式。

可以将此按键绑定视为 Emacs 的 C-x 1 绑定。

此按键绑定使用可绑定的 Readline 函数 tui-delete-other-windows.

C-x 2 ¶

使用至少包含两个窗口的 TUI 布局。如果当前布局已经有两个窗口，则使用下一个包含两个窗口的布局。 选择新布局时，总有一个窗口是上一个布局和新布局共有的。

可以将其视为 Emacs 的 C-x 2 绑定。

此按键绑定使用可绑定的 Readline 函数 tui-change-windows.

C-x o ¶

更改活动窗口。TUI 将几个按键绑定（如滚动和箭头键）与活动窗口关联。此命令将焦点切换到下一个 TUI 窗口。

可以将其视为 Emacs 的 C-x o 绑定。

此按键绑定使用可绑定的 Readline 函数 tui-other-window.

C-x s ¶

切换 TUI 单键模式的开启和关闭，该模式将单键绑定到 GDB 命令（请参阅 TUI 单键模式）。

此按键绑定使用可绑定的 Readline 函数 next-keymap.

以下按键绑定仅在 TUI 模式下有效：

PgUp ¶

将活动窗口向上滚动一页。

PgDn ¶

将活动窗口向下滚动一页。

上箭头 ¶

将活动窗口向上滚动一行。

下箭头 ¶

将活动窗口向下滚动一行。

左箭头 ¶

将活动窗口向左滚动一列。

右箭头 ¶

将活动窗口向右滚动一列。

C-L ¶

刷新屏幕。

由于在 TUI 模式下箭头键用于滚动活动窗口，因此除非命令窗口具有焦点，否则 readline 无法正常使用它们。当另一个窗口处于活动状态时，必须使用 其他 readline 按键绑定，如 C-p, C-n, C-b 和 C-f 来控制命令窗口。
