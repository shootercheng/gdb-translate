23 扩展 GDB ¶

GDB 提供了几种扩展机制。 GDB 还提供了在读取用于调试的文件时自动加载扩展的功能。这允许用户自动定制 GDB 针对正在调试的程序。

为了便于使用扩展语言， GDB 能够计算文件的内容。在这样做时， GDB 可以通过查看文件名扩展名来识别正在使用的扩展语言。文件名扩展名未被识别的文件总是被视为 GDB 命令文件。请参阅 命令文件.

你可以使用以下设置控制 GDB 如何计算这些文件：

set script-extension off ¶

所有脚本总是作为 GDB 命令文件进行计算。

set script-extension soft

调试器根据文件名扩展名确定脚本语言。如果支持这种脚本语言， GDB 使用该语言计算脚本。否则，它将文件作为 GDB 命令文件进行计算。

set script-extension strict

调试器根据文件名扩展名确定脚本语言，并使用该语言计算脚本。如果该语言不受支持，则计算失败。

show script-extension

显示 script-extension 选项的当前值。

  • 预定义的命令序列
  • 命令别名
  • 扩展 GDB 使用Python
  • 扩展 GDB 使用Guile
  • 自动加载扩展
  • 多种扩展语言
