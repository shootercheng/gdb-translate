21.1.6 达尔文系统 ¶

GDB 提供了以下特定于达尔文目标系统的命令：

set debug darwin num ¶

当设置为非零值时，启用特定于达尔文系统支持的调试消息。值越大，输出越详细。

show debug darwin ¶

显示达尔文系统消息的当前状态。

set debug mach-o num ¶

当设置为非零值时，在 GDB 读取达尔文系统目标文件时启用调试消息。（Mach - O 是达尔文系统上用于目标文件和可执行文件的文件格式。）值越大，输出越详细。这是一个用于诊断 GDB 内部问题的命令，正常使用时通常不需要。

show debug mach-o ¶

显示Mach - O文件消息的当前状态。

set mach-exceptions on ¶
set mach-exceptions off

在达尔文系统上，故障首先作为Mach异常报告，然后映射到Posix信号。使用此命令打开对下级进程中Mach异常的捕获。这有时可能有助于更好地理解故障的原因。默认情况下是关闭的。

show mach-exceptions ¶

显示异常捕获的当前状态。
