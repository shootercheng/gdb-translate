20.2 将文件发送到远程系统 ¶

一些远程目标提供了通过用于通信的同一连接传输文件的功能。 GDB 对于可通过其他方式访问的目标，例如 GNU运行的/Linux系统 gdbserver 通过网络接口。对于其他目标，例如只有单个串行端口的嵌入式设备，这可能是上传或下载文件的唯一方式。

并非所有远程目标都支持这些命令。

remote put hostfile targetfile ¶

复制文件 主机文件 从主机系统（运行的机器 GDB）到 目标文件 在目标系统上。

remote get targetfile hostfile ¶

复制文件 目标文件 从目标系统到 主机文件 在主机系统上。

remote delete targetfile ¶

删除 目标文件 从目标系统。
