23.3.2.19 Python 中的线程 ¶

GDB通过 gdb.InferiorThread 类。

以下与线程相关的函数在 gdb 模块中可用：

函数： gdb.selected_thread () ¶

此函数返回选定线程的线程对象。如果没有选定的线程，将返回 None.

要获取 inferior 的线程列表，请使用 Inferior.threads() 方法。请参阅 Python 中的 Inferiors.

一个 gdb.InferiorThread 对象具有以下属性：

变量： InferiorThread.name ¶

线程的名称。如果用户使用 thread name, 指定了名称，则返回该名称。否则，如果操作系统提供了名称，则返回该名称。否则，返回 None.

可以为该属性赋值。新值必须是字符串对象，用于设置新名称，或者 None, 用于删除用户指定的线程名称。

变量： InferiorThread.num ¶

由 GDB 分配的每个 inferior 中的线程编号。

变量： InferiorThread.global_num ¶

由 GDB 分配的线程全局 ID。例如，您可以使用此 ID 使 Python 断点特定于线程（请参阅 Breakpoint.thread 属性).

变量： InferiorThread.ptid ¶

由操作系统分配的线程 ID。此属性是一个包含三个整数的元组。第一个是进程 ID（PID）；第二个是轻量级进程 ID（LWPID），第三个是线程 ID（TID）。LWPID 或 TID 可能为 0，表示操作系统不使用该标识符。

变量： InferiorThread.ptid_string ¶

此只读属性包含一个表示 InferiorThread.ptid. 的字符串。这是 GDB 在‘目标 ID’列中使用的字符串，在 info threads 输出中（请参阅 ‘info threads’).

变量： InferiorThread.inferior ¶

此线程所属的 inferior。此属性表示为一个 gdb.Inferior 对象。此属性不可写。

变量： InferiorThread.details ¶

包含目标特定线程状态信息的字符串。此字符串的格式因目标而异。如果此线程没有额外的状态信息，则此属性包含 None.

例如，在 GNU/Linux 系统上，正在退出的线程将返回字符串‘正在退出’。对于远程目标，如果目标支持，将使用‘ details qThreadExtraInfo’远程数据包获取该字符串（请参阅‘ qThreadExtraInfo’).在

GDB ‘ details 目标 ID’列中显示该字符串，在输出中（请参阅 info threads ‘ info threads’’).

一个 gdb.InferiorThread 对象具有以下方法：

函数： InferiorThread.is_valid () ¶

返回 True 如果该 gdb.InferiorThread 对象有效， False 如果无效。一个 gdb.InferiorThread 对象在以下情况下将变为无效：线程退出，或者该线程所属的 inferior 被删除。所有其他 gdb.InferiorThread 方法在调用时如果对象无效，将抛出异常。

函数： InferiorThread.switch () ¶

这会将 GDB当前选定的线程更改为该对象所代表的线程。

函数： InferiorThread.is_stopped () ¶

返回一个布尔值，指示线程是否已停止。

函数： InferiorThread.is_running () ¶

返回一个布尔值，指示线程是否正在运行。

函数： InferiorThread.is_exited () ¶

返回一个布尔值，指示线程是否已退出。

函数： InferiorThread.handle () ¶

返回线程对象的句柄，以 Python bytes 对象表示。可以通过以下方式构造句柄的表示形式： gdb.Value 其中 gdb.Value(bufobj, type) bufobj 是句柄的 Python bytes 表示形式，并且 type 是句柄类型的 gdb.Type 。可以像在普通 Python 中一样为

对象添加任意属性。例如，如果需要对与线程相关的信息进行额外记录，这将很有用。 gdb.InferiorThread 请参阅

选择属性名称 , 以获取为新属性选择合适名称的指导。在这个人为构造的示例中，我们记录线程最后一次停止的时间： 

(gdb) python
import datetime

def thread_stopped(event):
    if event.inferior_thread is not None:
        thread = event.inferior_thread
    else:
        thread = gdb.selected_thread()
    thread._last_stop_time = datetime.datetime.today()

gdb.events.stop.connect(thread_stopped)
