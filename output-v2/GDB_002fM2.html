15.4.9.9 GDB 和Modula - 2 ¶

一些 GDB 命令在调试Modula - 2程序时用处不大。 以下五个子命令 set print 和 show print 专门适用于C和C++: ‘虚表’, ‘反汇编’, ‘汇编反汇编’, ‘对象’, 和 ‘联合’。前四个 适用于C++, 最后一个适用于C union 类型，而Modula - 2中没有与之直接 对应的类型。

该 @ 运算符（见 表达式），虽然在任何语言中都可用，但在Modula - 2中并无用处。其 目的是帮助调试 动态数组, 但Modula - 2不能像C或C 那样创建动态数组。 然而，由于可以用整数常量指定一个地址，所以构造 ‘++{类型}地址表达式’ 仍然有用。

在 GDB 脚本中，Modula - 2的不等运算符 # 被 解释为注释的开始。请使用 <> 代替。
