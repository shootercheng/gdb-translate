21.3.5 MicroBlaze ¶

MicroBlaze 是一种软核处理器，支持多种 Xilinx FPGA，如 Spartan 或 Virtex 系列。配备这些处理器的开发板通常有 JTAG 端口，可连接到运行 Xilinx 嵌入式开发套件 (EDK) 或软件开发套件 (SDK) 的主机系统。这个主机系统用于将配置比特流下载到目标 FPGA。Xilinx 微处理器调试器 (XMD) 程序通过 JTAG 接口与目标开发板通信，并提供一个 gdbserver 到开发板的接口。默认情况下 xmd 使用端口 1234. （虽然可以更改此默认端口，但需要使用未公开的 xmd 命令。如果需要这样做，请联系 Xilinx 技术支持。）

使用这些 GDB 命令连接到 MicroBlaze 目标处理器。

target remote :1234

如果您在与 GDB 相同的系统上运行 xmd.，则使用此命令连接到目标。

target remote xmd-host:1234

如果目标连接到 xmd 在名为 xmd - 主机. 的不同系统上运行，则使用此命令连接到目标。

load

使用此命令将程序下载到 MicroBlaze 目标。

set debug microblaze n

如果非零，则启用 MicroBlaze 特定的调试消息。

show debug microblaze n

显示 MicroBlaze 特定的调试级别。
