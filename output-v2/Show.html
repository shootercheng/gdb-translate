15.2 显示语言 ¶

以下命令可帮助您确定哪种语言是当前工作语言，以及源文件是用什么语言编写的。

show language

显示当前工作语言。这是您可以与诸如以下命令一起使用的语言 print 来构建和计算可能涉及程序中变量的表达式。

info frame ¶

显示此栈帧的源语言。如果您使用此栈帧中的标识符，此语言将成为工作语言。请参阅 关于栈帧的信息, 以识别此处列出的其他信息。

info source ¶

显示此源文件的源语言。请参阅 检查符号表, 以识别此处列出的其他信息。

在不寻常的情况下，您可能有扩展名不在标准列表中的源文件。然后，您可以显式设置与某种语言关联的扩展名：

set extension-language ext language ¶

告诉 GDB 扩展名为 ext 的源文件应被假定为用源语言 language编写。

info extensions ¶

列出所有文件名扩展名及其关联的语言。
