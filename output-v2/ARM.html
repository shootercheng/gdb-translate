21.3.2 ARM ¶

GDB 提供了以下特定于ARM的命令：

set arm disassembler ¶

此命令从反汇编风格列表中进行选择。 "std" 风格是标准风格。

show arm disassembler ¶

显示当前的反汇编风格。

set arm apcs32 ¶

此命令在32位和26位之间切换ARM操作模式。

show arm apcs32

显示ARM 32位模式的当前使用情况。

set arm fpu fputype

此命令设置ARM浮点单元（FPU）类型。参数 fputype 可以是以下之一：

auto

通过查询操作系统ABI来确定FPU类型。

softfpa

软件FPU，在小端ARM处理器上使用混合端序双精度浮点数。

fpa

GCC编译的FPA协处理器。

softvfp

使用纯端序双精度浮点数的软件FPU。

vfp

VFP协处理器。

show arm fpu

显示当前的FPU类型。

set arm abi

此命令强制 GDB 使用指定的ABI。

show arm abi

显示当前使用的ABI。

set arm fallback-mode (arm|thumb|auto)

GDB 在可用时使用符号表来确定指令是ARM还是Thumb。此命令控制 GDB在符号表不可用时的默认行为。默认值是‘自动’，这会使 GDB 使用当前执行模式（从 T 位在 CPSR 寄存器中）。

show arm fallback-mode

显示当前的备用指令模式。

set arm force-mode (arm|thumb|auto)

此命令覆盖使用符号表来确定指令是ARM还是Thumb。默认值是‘自动’，这会使 GDB 使用符号表，然后使用‘设置arm备用模式’的设置。

show arm force-mode

显示当前的强制指令模式。

set arm unwind-secure-frames

此命令启用在具有安全扩展的Cortex - M上从非安全模式到安全模式的回溯。 在回溯异常栈时，这可能会触发安全异常。 默认情况下是启用的。

show arm unwind-secure-frames

显示是否启用了从非安全模式到安全模式的回溯。

set debug arm

切换是否显示来自ARM目标支持子系统的特定于ARM的调试消息。

show debug arm

显示是否启用了特定于ARM的调试消息。

target sim [simargs] …

GDB ARM模拟器接受以下可选参数。

--swi-support=type

告诉模拟器支持哪些SWI接口。参数 类型 可以是以下值的逗号分隔列表。 默认值是 all.

none
demon
angel
redboot
all
