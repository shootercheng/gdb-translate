15.4.9.3 常量 ¶

GDB 允许你以以下方式表示 Modula - 2 的常量：

  • 整数常量只是一串数字。在表达式中使用时，常量会被解释为与表达式的其他部分类型兼容。十六进制整数通过尾随的 ‘H’ 来指定，八进制整数通过尾随的 ‘B’ 来指定。
  • 浮点常量表现为一串数字，后面跟着一个小数点和另一串数字。然后可以指定一个可选的指数，形式为 ‘E[+|-]nnn’，其中 ‘[+|-]nnn’ 是所需的指数。浮点常量的所有数字都必须是有效的十进制（基数为 10）数字。
  • 字符常量由一对相同的引号（单引号 (') 或双引号 (")）括起来的单个字符组成。它们也可以用它们的序号值（通常是它们的 ASCII 值）后面跟着一个 ‘C’ 来表示。
  • 字符串常量由一对相同的引号（单引号 (') 或双引号 (")）括起来的一串字符组成。也允许使用 C 风格的转义序列。请参阅 C 和 C++ 常量, 以简要了解转义序列。
  • 枚举常量由一个枚举标识符组成。
  • 布尔常量由标识符 TRUE 和 FALSE组成。
  • 指针常量仅由整数值组成。
  • 集合常量目前尚不支持。
