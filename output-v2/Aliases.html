23.2 命令别名 ¶

别名允许你为现有命令定义替代拼写。 例如，如果一个用Python定义的新 GDB 命令（请参阅 扩展 GDB 使用Python）有一个很长的名称，那么有一个缩写版本会很方便，这样可以减少输入量。

GDB 它本身就使用了别名。例如，‘s’ 是 ‘step’ 命令的别名，尽管它也是其他命令（如 ‘set’ 和 ‘show’）的模糊缩写。

别名还用于为多字命令提供缩短或更常用的版本。例如， GDB 提供了 ‘tty’ 作为 ‘set inferior-tty’ 命令的别名。

你可以使用 ‘alias’ 命令定义一个新的别名。

alias [-a] [--] alias = command [default-args] ¶

alias 指定新别名的名称。‘ alias ’ 的每个单词必须由字母、数字、连字符和下划线组成。

command 指定被别名化的现有命令的名称。

command 也可以是现有别名的名称。在这种情况下， command 不能是有默认参数的别名。

‘-a’ 选项指定新别名是命令的缩写。缩写在命令补全中不使用。

‘--’ 选项指定选项的结束，当 alias 以连字符开头时很有用。

你可以为你的别名指定 默认参数 。这些 默认参数 将自动添加到在命令行上显式输入的别名参数之前。

例如，下面定义了一个别名 btfullall ，用于显示所有局部变量和所有帧参数： 

(gdb) alias btfullall = backtrace -full -frame-arguments all

有关 默认参数, 请参阅 默认参数。

这里有一个简单的示例，展示了如何为一个命令创建缩写，以减少输入量。假设你厌倦了输入 ‘disas’，它是 ‘disassemble’ 命令当前最短的无歧义缩写，你想要一个更短的版本，名为 ‘di’。以下操作可以实现这一点。 

(gdb) alias -a di = disas

请注意，别名与用户定义的命令不同。对于用户定义的命令，你还需要使用 ‘document’ 命令为其编写文档。别名会自动采用现有命令的文档。

这里有一个示例，我们将 ‘elms’ 作为 ‘elements’ 在 ‘set print elements’ 命令中的缩写。 这表明你可以为命令的任何部分创建缩写。 

(gdb) alias -a set print elms = set print elements
(gdb) alias -a show print elms = show print elements
(gdb) set p elms 200
(gdb) show p elms
打印字符串字符或数组元素的限制是200。

请注意，如果你正在定义一个 ‘set’ 命令的别名，并且你想为相应的 ‘show’ 命令创建一个别名，那么你需要单独定义后者。

无歧义的缩写命令在 command 和 alias, 中是允许的，就像通常一样。 

(gdb) alias -a set pr elms = set p ele

最后，这里有一个示例，展示了如何为一个更复杂的命令创建一个单字别名。 这将创建命令 ‘spe’，它是命令 ‘set print elements’ 的别名。 

(gdb) alias spe = set print elements
(gdb) spe 20
