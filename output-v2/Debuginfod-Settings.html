K.1 Debuginfod 设置 ¶

GDB 提供了以下用于配置的命令 debuginfod.

set debuginfod enabled ¶
set debuginfod enabled on

GDB 可以查询 debuginfod 服务器以获取缺失的调试信息和源文件。 GDB 还可以下载单个 ELF/DWARF 节，例如 .gdb_index 以帮助减少从 debuginfod 服务器下载的总数据量；这可以通过 maint set debuginfod download-sections （请参阅 maint set debuginfod download-sections)。

set debuginfod enabled off

GDB 在缺少调试信息或源文件时不会尝试查询 debuginfod 服务器。默认情况下， debuginfod enabled 在非交互式会话中设置为 off 。

set debuginfod enabled ask

GDB 在尝试执行下一次查询之前会提示用户启用或禁用 debuginfod 。默认情况下， debuginfod enabled 在交互式会话中设置为 ask 。

show debuginfod enabled ¶

显示是否将 debuginfod enabled 设置为 on, off 或 ask。

set debuginfod urls ¶
set debuginfod urls urls

设置 debuginfod 将尝试查询的以空格分隔的 URL 列表。仅应使用 http://, https:// 和 file:// 协议。 debuginfod urls 的默认值从 DEBUGINFOD_URLS 环境变量复制而来。

show debuginfod urls ¶

显示 debuginfod 将尝试查询的 URL 列表。

set debuginfod verbose ¶
set debuginfod verbose n

启用或禁用 debuginfod相关的输出。使用非零值启用，使用 0 禁用。 debuginfod 默认情况下会显示输出。

show debuginfod verbose ¶

显示当前的详细程度设置。
