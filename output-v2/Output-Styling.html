22.5 输出样式设置

set style enabled ‘on|off’

启用或禁用所有样式设置。默认值取决于主机，大多数主机默认为 ‘开启’。

NO_COLOR 环境变量设置为非空值时，那么 GDB 将在启动时将其更改为 ‘关闭’。

show style enabled

显示当前样式设置状态。

set style emoji ‘auto|on|off’

启用或禁用表情符号的使用。在大多数主机上，默认值为 ‘自动’，这意味着只有当主机字符集为 ‘UTF - 8’ 时才会使用表情符号；然而，在 Windows 上使用控制台时，默认值为 ‘关闭’。请注意，完全禁用样式设置也会阻止表情符号显示。

目前，每当 GDB 报告错误或警告时都会打印表情符号。

show style emoji

显示当前表情符号输出状态。

set style sources ‘on|off’

启用或禁用源代码样式设置。这会影响源代码（例如 list 命令的输出）是否进行样式设置。默认值为 ‘开启’。请注意，源代码样式设置仅在一般样式设置启用且有可用的源代码高亮库时才有效。 GDB。

有两种进行高亮显示的方法。首先，如果 GDB 与 GNU 源代码高亮库链接，则使用该库。否则，如果 GDB 配置了 Python 脚本支持，并且 Python Pygments 包可用，则将使用该包。

show style sources

显示当前源代码样式设置状态。

set style warning-prefix
show style warning-prefix
set style error-prefix
show style error-prefix

这些命令分别控制在警告和错误之前打印的前缀。此功能旨在与表情符号显示一起使用，因此只有在启用表情符号样式设置时才会显示前缀。默认情况下，警告使用警告符号表情符号，错误使用叉号表情符号。

set style tui-current-position ‘on|off’

启用或禁用 TUI 当前位置指示器突出显示的源代码和汇编代码的样式设置。默认值为 ‘关闭’。 请参阅 GDB 文本用户界面。

show style tui-current-position

显示 TUI 当前位置指示器突出显示的源代码和汇编代码是否进行了样式设置。

set style disassembler enabled ‘on|off’

启用或禁用反汇编样式设置。这会影响反汇编输出（例如 disassemble 命令的输出）是否进行样式设置。反汇编样式设置仅在一般样式设置启用（使用 set style enabled on）且有可用的源代码高亮库时才有效。 GDB。

可用于设置反汇编输出样式的两个源代码高亮库是： GDB GDB的内置反汇编器，或 Python Pygments 包。

GDB首选使用内置反汇编器进行样式设置，这通常能提供更好的效果，因为它能够对不同类型的指令操作数进行不同的样式设置。然而，内置反汇编器并非能对所有架构进行样式设置。

对于内置反汇编器无法进行样式设置的架构， GDB 将尽可能回退到使用 Python Pygments 包。为了使用 Python Pygments 包， GDB 必须使用 Python 支持进行构建，并且必须安装 Pygments 包。

如果这两个选项都不可用，那么 GDB 将生成无样式的反汇编输出，即使此设置为 ‘开启’。

要确定当前架构是否支持使用内置反汇编库进行样式设置，请参阅 维护 显示 libopcodes - 样式设置 启用。

show style disassembler enabled

显示当前反汇编样式设置状态。

set style filename background color

将背景色设置为 颜色。 颜色 可以是 ‘无’ （表示终端的默认颜色）、ISO/IEC 6429 八种标准颜色之一的名称、终端调色板中 0 到 255 的索引，或者 24 位真彩色的 ‘#RRGGBB’ 格式的十六进制 RGB 三元组。

有效的颜色名称有 ‘黑色’、‘红色’、‘绿色’、 ‘黄色’、‘蓝色’、‘品红色’、‘青色’ 和 ‘白色’。

整数 0 到 7 是标准颜色的同义词。整数 8 - 15 用于 aixterm 扩展 16 色调色板中的所谓亮色。整数 16 - 255 是 xterm 扩展 256 色调色板的索引（通常是 6x6x6 立方体加上灰度渐变）。一般来说，256 色调色板依赖于终端，有时可以使用 OSC 4 序列进行更改，例如 “\033]4;1;rgb:00/FF/00\033\\”。十六进制 24 位真彩色以格式 ‘#RRGGBB’ 指定，其中 RR、GG 和 BB 分别是指定红、绿、蓝颜色分量强度的两位十六进制整数。

用户有责任验证终端是否支持指定的颜色。

set style filename foreground color

将前景色设置为 颜色。 颜色 的指定方式与背景色相同。

set style filename intensity value

将强度设置为 值。有效的强度值有 ‘正常’ （默认值）、‘粗体’ 和 ‘暗淡’。

该 show style 命令及其子命令会使用自身的样式对输出中的样式名称进行样式设置。 因此，使用 show style 查看完整的样式列表、它们的特性以及每种样式的视觉效果。

可设置样式的对象有：

command

控制任何 GDB 命令显示的样式。默认情况下，此样式的强度为粗体。 GDB控制文件名和 URL 的样式。默认情况下，此样式的前景色为绿色。

filename

控制函数名的样式。这些由

function

命令族管理。默认情况下，此样式的前景色为黄色。 set style function 如果

在使用其内置反汇编库进行样式设置时，此样式也用于样式化反汇编输出中的符号名称（请参阅 GDB 设置样式 反汇编 启用 ）。

line-number

控制行号的样式。默认情况下，此样式的强度为暗淡。

variable

控制变量名的样式。这些由 set style variable 命令族管理。默认情况下，此样式的前景色为青色。

address

控制地址的样式。这些由 set style address 命令族管理。默认情况下，此样式的前景色为蓝色。

如果 GDB 在使用其内置反汇编库进行样式设置时，此样式也用于样式化反汇编输出中的地址（请参阅 设置样式 反汇编 启用）。

version

控制 GDB版本号文本的样式。默认情况下，此样式的前景色为品红色，强度为粗体。版本号在两个地方显示， show version的输出中，以及 GDB 启动时。

为了控制 GDB 在启动时如何设置版本号的样式，将 set style version 命令族添加到早期初始化命令文件中（请参阅 初始化文件）。

title

控制标题的样式。这些由 set style title 命令族管理。默认情况下，此样式的强度为粗体。标题样式在显示列表的标题行时使用，例如 info breakpoints 输出的第一行（请参阅 设置断点）。

highlight

控制高亮显示的样式。这些由 set style highlight 命令族管理。默认情况下，此样式的前景色为红色。命令使用高亮样式来吸引用户对其输出中某些特定部分的注意。例如，命令 apropos -v REGEXP 使用高亮样式标记与 正则表达式。

metadata

控制 GDB 添加到显示数据中的数据注释的样式。默认情况下，此样式的强度为暗淡。元数据注释包括 ‘重复 n 次’ 注释，用于抑制重复数组元素的显示（请参阅 打印设置）、‘<不可用>’ 和 ‘<错误 描述>’ 错误注释以及 ‘<优化掉>’ 注释，用于在回溯中显示栈帧信息时表示已优化掉的值（请参阅 回溯）等。

tui-border

控制 TUI 边框的样式。请注意，与其他样式选项不同，只能通过 set style控制边框的颜色。这样做是出于兼容性原因，因为设置边框强度的 TUI 控件早于将通用样式添加到 GDB之前就存在了。请参阅 TUI 配置变量。

tui-active-border

控制活动 TUI 边框的样式；即具有焦点的 TUI 窗口的样式。

disassembler comment

控制反汇编输出中注释的样式。这些由 set style disassembler comment 命令族管理。此样式仅在 GDB 使用其内置反汇编库进行样式设置时使用（请参阅 设置样式 反汇编 启用）。默认情况下，此样式的强度为暗淡，前景色为白色。

disassembler immediate

控制反汇编输出中数字操作数的样式。这些由 set style disassembler immediate 命令族管理。此样式不适用于表示地址的指令操作数，在这种情况下，将使用 ‘反汇编地址’ 样式。此样式仅在 GDB 使用其内置反汇编库进行样式设置时使用。默认情况下，此样式的前景色为蓝色。

disassembler address

控制反汇编输出中地址操作数的样式。这是 ‘地址’ 样式的别名。

disassembler symbol

控制反汇编输出中符号名称的样式。这是 ‘函数’ 样式的别名。

disassembler mnemonic

控制反汇编输出中指令助记符的样式。这些由 set style disassembler mnemonic 命令族管理。此样式也用于汇编指令，例如 .byte, .word, 等。此样式仅在 GDB 使用其内置反汇编库进行样式设置时使用。默认情况下，此样式的前景色为绿色。

disassembler register

控制反汇编输出中寄存器操作数的样式。这些由 set style disassembler register 命令族管理。此样式仅在 GDB 使用其内置反汇编库进行样式设置时使用。默认情况下，此样式的前景色为红色。
