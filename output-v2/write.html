写入 ¶

概要：
int write(int fd, const void *buf, unsigned int count);
请求：

‘Fwrite,文件描述符,缓冲区指针,数量’

返回值：

成功时，返回写入的字节数。 返回零表示没有写入任何内容。出错时，返回 -1。

错误：
EBADF

文件描述符 不是有效的文件描述符，或者未以写入模式打开。

EFAULT

缓冲区指针 是无效的指针值。

EFBIG

尝试写入的文件大小超过了主机特定的最大文件大小限制。

ENOSPC

设备上没有空间来写入数据。

EINTR

调用被用户中断。
