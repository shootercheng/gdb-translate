10.10.1 漂亮打印机介绍 ¶

当 GDB 打印一个值时，它首先会查看是否有为该值注册的漂亮打印机。如果有，那么 GDB 会调用该漂亮打印机来打印该值。否则，该值将正常打印。

漂亮打印机通常都有名称。这使得它们易于管理。‘info pretty-printer’命令将列出所有已安装的漂亮打印机及其名称。 如果一个漂亮打印机可以处理多种数据类型，那么它的 子打印机 就是针对各个数据类型的打印机。 每个这样的子打印机都有自己的名称。 名称的格式是 打印机名称;子打印机名称.

漂亮打印机通过 注册 到 GDB来安装。 通常，当相应的调试信息被加载时，它们会自动加载并注册，这样就无需进行任何特殊操作即可使用它们。

漂亮打印机可以在三个地方进行注册。

  • 全局注册的漂亮打印机在调试所有下级进程时都可用。
  • 在程序空间中注册的漂亮打印机仅在调试该程序时可用。 请参阅 Python 中的程序空间, 以获取有关 Python 中程序空间的更多详细信息。
  • 在目标文件中注册的漂亮打印机将随相应的目标文件（例如，共享库）一起加载和卸载。 请参阅 Python 中的目标文件, 以获取有关 Python 中目标文件的更多详细信息。

请参阅 选择漂亮打印机, 以获取有关如何选择漂亮打印机的更多信息。

请参阅 编写漂亮打印机, 以了解如何为新类型实现漂亮打印机。
