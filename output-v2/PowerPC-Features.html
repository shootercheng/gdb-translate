G.5.12 PowerPC 特性 ¶

‘org.gnu.gdb.power.core’ 特性是 PowerPC 目标所必需的。它应该包含寄存器 ‘r0’ 到 ‘r31’、 ‘pc’、‘msr’、‘cr’、‘lr’、‘ctr’，以及 ‘xer’。它们可能是 32 位或 64 位，具体取决于目标。

‘org.gnu.gdb.power.fpu’ 特性是可选的。它应该包含寄存器 ‘f0’ 到 ‘f31’ 和 ‘fpscr’。

‘org.gnu.gdb.power.altivec’ 特性是可选的。它应该包含寄存器 ‘vr0’ 到 ‘vr31’、‘vscr’，以及 ‘vrsave’。 GDB 将定义伪寄存器 ‘v0’ 到 ‘v31’ 作为相应 ‘vrX’ 寄存器的别名。

‘org.gnu.gdb.power.vsx’ 特性是可选的。它应该包含寄存器 ‘vs0h’ 到 ‘vs31h’。 GDB 将 这些寄存器与浮点寄存器（‘f0’ 到 ‘f31’）和 Altivec 寄存器（‘vr0’ 到 ‘vr31’）组合起来，以展示 128 位宽的寄存器 ‘vs0’ 到 ‘vs63’，即 POWER7 的向量 - 标量寄存器集。 因此，此特性需要 ‘org.gnu.gdb.power.fpu’ 和 ‘org.gnu.gdb.power.altivec’。

‘org.gnu.gdb.power.spe’ 特性是可选的。它应该包含寄存器 ‘ev0h’ 到 ‘ev31h’、‘acc’，以及 ‘spefscr’。SPE 目标应该在 ‘org.gnu.gdb.power.core’ 中提供 32 位寄存器，并在 ‘ev0h’ 到 ‘ev31h’。 GDB 将这些组合起来，向用户展示寄存器 ‘ev0’ 到 ‘ev31’。

‘org.gnu.gdb.power.ppr’ 特性是可选的。它应该包含 64 位寄存器 ‘ppr’。

‘org.gnu.gdb.power.dscr’ 特性是可选的。它应该包含 64 位寄存器 ‘dscr’。

‘org.gnu.gdb.power.tar’ 特性是可选的。它应该包含 64 位寄存器 ‘tar’。

‘org.gnu.gdb.power.ebb’ 特性是可选的。它应该包含寄存器 ‘bescr’、‘ebbhr’ 和 ‘ebbrr’，所有这些都是 64 位宽。

‘org.gnu.gdb.power.linux.pmu’ 特性是可选的。它应该包含寄存器 ‘mmcr0’、‘mmcr2’、‘siar’、‘sdar’ 和 ‘sier’，所有这些都是 64 位宽。这是由 GNU/Linux 提供的 isa 2.07 服务器 PMU 寄存器的子集。

‘org.gnu.gdb.power.htm.spr’ 特性是可选的。它应该包含寄存器 ‘tfhar’、‘texasr’ 和 ‘tfiar’，所有这些都是 64 位宽。

‘org.gnu.gdb.power.htm.core’ 特性是可选的。它应该包含检查点通用寄存器 ‘cr0’ 到 ‘cr31’，以及检查点寄存器 ‘clr’ 和 ‘cctr’。这些寄存器可能是 32 位或 64 位，具体取决于目标。它还应该包含检查点寄存器 ‘ccr’ 和 ‘cxer’，这两个寄存器都应该是 32 位宽。

‘org.gnu.gdb.power.htm.fpu’ 特性是可选的。它应该包含检查点 64 位浮点寄存器 ‘cf0’ 到 ‘cf31’，以及检查点 64 位寄存器 ‘cfpscr’。

‘org.gnu.gdb.power.htm.altivec’ 特性是可选的。它 应该包含检查点 Altivec 寄存器 ‘cvr0’ 到 ‘cvr31’，所有这些都是 128 位宽。它还应该包含检查点寄存器 ‘cvscr’ 和 ‘cvrsave’，这两个都是 32 位宽。

‘org.gnu.gdb.power.htm.vsx’ 特性是可选的。它应该包含寄存器 ‘cvs0h’ 到 ‘cvs31h’。 GDB 将这些寄存器与检查点浮点寄存器（‘cf0’ 到 ‘cf31’）和检查点 Altivec 寄存器（‘cvr0’ 到 ‘cvr31’）组合起来，以展示 128 位宽的检查点向量 - 标量寄存器 ‘cvs0’ 到 ‘cvs63’。因此，此特性需要 ‘org.gnu.gdb.power.htm.altivec’ 和 ‘org.gnu.gdb.power.htm.fpu’。

‘org.gnu.gdb.power.htm.ppr’ 特性是可选的。它应该包含 64 位检查点寄存器 ‘cppr’。

‘org.gnu.gdb.power.htm.dscr’ 特性是可选的。它应该包含 64 位检查点寄存器 ‘cdscr’。

‘org.gnu.gdb.power.htm.tar’ 特性是可选的。它应该包含 64 位检查点寄存器 ‘ctar’。
