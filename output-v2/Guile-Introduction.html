23.4.1 Guile 介绍 ¶

Guile 是 Scheme 编程语言的一种实现，并且是 GNU 项目的官方扩展语言。

对 Guile 的支持在 GDB 中与对 Python 的支持在 GDB 中相当接近，因此那里的概念应该可以沿用。不过，在合理的情况下，有些事情的处理方式会有所不同。

GDB 需要 Guile 版本 3.0、2.2 或 2.0。

使用的 Guile 脚本 GDB 应该安装在 数据目录/guile, 其中 数据目录 是在 GDB 启动时确定的数据目录（请参阅 GDB 数据文件)。这个目录，称为 Guile 目录，会自动添加到 Guile 搜索路径中，以便 Guile 解释器能够找到安装在该位置的所有脚本。
