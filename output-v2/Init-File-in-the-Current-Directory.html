22.8.1 自动加载当前目录中的初始化文件 ¶

默认情况下， GDB 会从当前工作目录中的初始化文件（如果有）读取并执行预定义的命令序列， 请参阅 启动期间当前目录中的初始化文件.

请注意，加载此本地 .gdbinit 文件还需要相应配置的 auto-load safe-path （请参阅 自动加载的安全限制).

set auto-load local-gdbinit [on|off] ¶

启用或禁用在当前目录的初始化文件中找到的预定义命令序列的自动加载 （请参阅 预定义的命令序列)。

show auto-load local-gdbinit ¶

显示是否启用了从当前目录的初始化文件中自动加载预定义命令序列。

info auto-load local-gdbinit ¶

打印是否已自动加载当前目录的初始化文件中的预定义命令序列。
