15.4 支持的语言 ¶

GDB 支持 C、C++++, D、Go、Objective - C、Fortran、 OpenCL C、Pascal、Rust、汇编、Modula - 2 和 Ada。 某些 GDB 特性可在表达式中使用，无论你使用何种 语言： GDB @ 和 :: 运算符， 以及 ‘{类型}地址’ 构造（参见 表达式) 可与任何支持的 语言的构造一起使用。

以下各节详细说明了每种源语言在何种程度上得到了 GDB的支持。 这些章节并非旨在作为语言 教程或参考资料，而仅作为参考指南，说明 GDB 表达式解析器接受什么，以及不同语言的输入和输出 格式应该是什么样的。 关于这些语言中的每一种都有很多优秀的 书籍；请参考这些书籍获取语言参考或教程。

  • C 和 C++++
  • D
  • Go
  • Objective - C
  • OpenCL C
  • Fortran
  • Pascal
  • Rust
  • Modula - 2
  • Ada
