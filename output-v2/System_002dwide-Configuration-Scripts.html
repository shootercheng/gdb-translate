C.6.1 已安装的系统范围配置脚本 ¶

system - gdbinit 目录，位于数据目录内 （如在配置时由 --with - gdb - datadir 指定）包含 多个可作为系统范围初始化文件使用的脚本。要在启动时自动加载这些脚本， GDB 应使用 --with - system - gdbinit进行配置。否则，任何用户都应该能够根据需要手动加载它们。

目前可用的脚本如下：

  • elinos.py 此脚本在调试 ELinOS 目标上的程序时很有用。 它利用标准 ELinOS 环境中定义的环境变量来确定系统共享库的位置，然后适当地设置 ‘solib - absolute - prefix’ 和 ‘solib - search - path’ 变量。
  • wrs - linux.py 此脚本在调试运行 Wind River Linux 的目标上的程序时很有用。它期望 ENV_PREFIX 设置为目标系统使用的主机端根文件系统。
