13.1.3 跟踪点通过计数 ¶

passcount [n [num]] ¶

设置 通过计数 跟踪点的通过计数。通过计数是一种自动停止跟踪实验的方法。如果一个跟踪点的通过计数是 n, 那么跟踪实验将在该跟踪点第 n次被触发时自动停止。如果未指定跟踪点编号 num ，该 passcount 命令将设置最近定义的跟踪点的通过计数。如果未给出通过计数，跟踪实验将运行直到用户明确停止。

示例： 

(gdb) 通过计数 5 2 // 在第5次执行时停止

                                   // tracepoint 2


(gdb) 通过计数 12  // 在第12次执行时停止

                                   // most recently defined tracepoint.

(gdb) 跟踪 foo
(gdb) 通过 3
(gdb) 跟踪 bar
(gdb) 通过 2
(gdb) 跟踪 baz
(gdb) 通过 1        // 当 foo 被执行时停止跟踪

                                    // executed 3 times OR when bar has

                                    // been executed 2 times

                                    // OR when baz has been executed 1 time.
