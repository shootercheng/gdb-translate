8.6 帧过滤器的管理。 ¶

帧过滤器是基于Python的实用工具，用于管理和装饰帧的输出。请参阅 过滤帧，以获取更多信息。

管理帧过滤器是通过 GDB中可用的几个命令来执行的，这里将详细介绍。

info frame-filter ¶

打印所有字典中已安装的帧过滤器列表，显示它们的名称、优先级和启用状态。

disable frame-filter filter-dictionary filter-name

禁用与 过滤器字典 和 过滤器名称匹配的字典中的帧过滤器。 过滤器字典 可以是 all, global, progspace, 或者是帧过滤器字典所在的目标文件的名称。当 all 被指定时，所有字典中的所有帧过滤器都将被禁用。 过滤器名称 是帧过滤器的名称，当 all 不是 过滤器字典的选项时使用。禁用的帧过滤器不会被删除，以后可以再次启用。

enable frame-filter filter-dictionary filter-name ¶

启用与 过滤器字典 和 过滤器名称匹配的字典中的帧过滤器。 过滤器字典 可以是 all, global, progspace 或者是帧过滤器字典所在的目标文件的名称。当 all 被指定时，所有字典中的所有帧过滤器都将被启用。 过滤器名称 是帧过滤器的名称，当 all 不是 过滤器字典的选项时使用。

示例： 

(gdb) info frame-filter

global frame-filters:
  Priority  Enabled  Name
  1000      No       PrimaryFunctionFilter
  100       Yes      Reverse

progspace /build/test frame-filters:
  Priority  Enabled  Name
  100       Yes      ProgspaceFilter

objfile /build/test frame-filters:
  Priority  Enabled  Name
  999       Yes      BuildProgramFilter

(gdb) disable frame-filter /build/test BuildProgramFilter
(gdb) info frame-filter

global frame-filters:
  Priority  Enabled  Name
  1000      No       PrimaryFunctionFilter
  100       Yes      Reverse

progspace /build/test frame-filters:
  Priority  Enabled  Name
  100       Yes      ProgspaceFilter

objfile /build/test frame-filters:
  Priority  Enabled  Name
  999       No       BuildProgramFilter

(gdb) enable frame-filter global PrimaryFunctionFilter
(gdb) info frame-filter

global frame-filters:
  Priority  Enabled  Name
  1000      Yes      PrimaryFunctionFilter
  100       Yes      Reverse

progspace /build/test frame-filters:
  Priority  Enabled  Name
  100       Yes      ProgspaceFilter

objfile /build/test frame-filters:
  Priority  Enabled  Name
  999       No       BuildProgramFilter
set frame-filter priority filter-dictionary filter-name priority ¶

设置与 优先级 匹配的字典中帧过滤器的 过滤器字典，以及与帧过滤器名称匹配的 过滤器名称。 过滤器字典 可以是 global, progspace 或者是帧过滤器字典所在的目标文件的名称。 优先级 是一个整数。

show frame-filter priority filter-dictionary filter-name ¶

显示与 优先级 匹配的字典中帧过滤器的 过滤器字典，以及与帧过滤器名称匹配的 过滤器名称。 过滤器字典 可以是 global, progspace 或者是帧过滤器字典所在的目标文件的名称。

示例： 

(gdb) info frame-filter

global frame-filters:
  Priority  Enabled  Name
  1000      Yes      PrimaryFunctionFilter
  100       Yes      Reverse

progspace /build/test frame-filters:
  Priority  Enabled  Name
  100       Yes      ProgspaceFilter

objfile /build/test frame-filters:
  Priority  Enabled  Name
  999       No       BuildProgramFilter

(gdb) set frame-filter priority global Reverse 50
(gdb) info frame-filter

global frame-filters:
  Priority  Enabled  Name
  1000      Yes      PrimaryFunctionFilter
  50        Yes      Reverse

progspace /build/test frame-filters:
  Priority  Enabled  Name
  100       Yes      ProgspaceFilter

objfile /build/test frame-filters:
  Priority  Enabled  Name
  999       No       BuildProgramFilter
