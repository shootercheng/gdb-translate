变量： Type.is_scalar ¶

如果该类型是标量类型，此属性为 True ，否则，此属性为 False。非标量类型的示例包括结构体、联合体和类。

变量： Type.is_signed ¶

对于标量类型（即那些 Type.is_scalar 为 True的类型），如果该类型是有符号的，此属性为 True ，否则此属性为 False。

尝试读取非标量类型（即 Type.is_scalar 为 False的类型）的此属性将引发一个 ValueError。

变量： Type.is_array_like ¶

一个布尔值，指示该类型是否类似数组。

某些语言具有在内部表示为结构体的类似数组的对象。例如，Rust 的切片类型或 Ada 的无约束数组就是这种情况。 GDB 可能了解这些类型。此判断是基于类型所源自的语言进行的。

变量： Type.is_string_like ¶

一个布尔值，指示该类型是否类似字符串。与 Type.is_array_like一样，这是根据类型的源语言来确定的。
