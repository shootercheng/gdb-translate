23.3.2.11 装饰帧 ¶

帧装饰器是帧过滤器的姊妹对象（参见 过滤帧）。帧装饰器由帧过滤器应用，并且只能与帧过滤器结合使用。

帧装饰器的目的是自定义每个 gdb.Frame 在执行帧过滤器的命令中的打印内容。 这个概念称为装饰帧。帧装饰器使用每个 API 调用中包含的 Python 代码来装饰一个 gdb.Frame 。 这将实际包含在一个 gdb.Frame 中的数据与帧装饰器产生的装饰后的数据分开。这种抽象对于维护每个 gdb.Frame中包含的数据的完整性是必要的。

帧装饰器有一个强制接口，定义如下。

GDB 已经包含一个名为 FrameDecorator的帧装饰器。这包含大量的 样板代码来装饰一个 gdb.Frame的内容。建议其他帧装饰器继承并扩展这个 对象，并且只覆盖所需的方法。

FrameDecorator 在 Python 模块 gdb.FrameDecorator中定义，因此您的代码可以像这样导入它： 

from gdb.FrameDecorator import FrameDecorator
函数： FrameDecorator.elided (self) ¶

该 elided 方法在分层系统中将帧分组。例如，在解释器中，多个低级 帧组成解释语言中的单个调用。在这个 示例中，帧过滤器将省略低级帧，并向用户呈现一个代表解释语言中调用的单个高级帧。

该 elided 函数必须返回一个可迭代对象，并且这个可迭代对象 必须包含被省略的帧，这些帧被包装在一个合适的 帧装饰器中。如果没有帧被省略，这个函数可以 返回一个空的可迭代对象，或 None. 被省略的帧在 CLI 回溯中相对于正常帧缩进，或者在 GDB/MI的情况下，被放置在省略帧的 children 字段中。

帧过滤器的任务还包括从源迭代器中过滤掉被省略的帧。这将避免重复打印帧。

函数： FrameDecorator.function (self) ¶

此方法返回要打印的帧中的函数名称。

此方法必须返回一个描述函数的 Python 字符串，或 None.

如果此函数返回 None, GDB 将不会为此字段打印任何 数据。

函数： FrameDecorator.address (self) ¶

此方法返回要打印的帧的地址。

此方法必须返回一个足够大的 Python 数值整数类型来描述帧的地址，或 None.

如果此函数返回一个 None, GDB 将不会为此字段打印任何 数据。

函数： FrameDecorator.filename (self) ¶

此方法返回与此帧关联的文件名和路径。

此方法必须返回一个包含文件名和 帧所依赖的目标文件路径的 Python 字符串，或 None.

如果此函数返回一个 None, GDB 将不会为此字段打印任何 数据。

函数： FrameDecorator.line (self): ¶

此方法返回与此帧所指向的函数内当前位置关联的行号。

此方法必须返回一个 Python 整数类型，或 None.

如果此函数返回一个 None, GDB 将不会为此字段打印任何 数据。

函数： FrameDecorator.frame_args (self) ¶

此方法必须返回一个可迭代对象，或 None. 返回一个 空的可迭代对象，或 None 意味着不会为该帧打印帧参数。这个可迭代对象必须包含实现这里描述的两个方法的对象。

这个对象必须实现一个 symbol 方法，该方法接受一个 单一的 self 参数，并且必须返回一个 gdb.Symbol （参见 符号的 Python 表示），或一个 Python 字符串。该对象还必须 实现一个 value 方法，该方法接受一个单一的 self 参数，并且必须返回一个 gdb.Value （参见 来自被调试程序的值）、一个 Python 值，或 None. 如果 value 方法返回 None, 并且 argument 方法返回一个 gdb.Symbol, GDB 将查找并打印 该 gdb.Symbol 的值。

一个简短的示例： 

class SymValueWrapper():

    def __init__(self, symbol, value):
        self.sym = symbol
        self.val = value

    def value(self):
        return self.val

    def symbol(self):
        return self.sym

函数： FrameDecorator.frame_locals (self) ¶

此方法必须返回一个可迭代对象或 None. 返回一个 空的可迭代对象，或 None 意味着不会为该帧打印帧局部参数。

对象接口、读取帧局部变量的各种策略的描述以及示例与 在 frame_args 函数中描述的内容基本相似（参见 帧过滤器的 frame_args 函数）。以下是一个修改后的示例：

    def frame_locals(self):
        vars = []
        try:

        except:

        # 添加一个合成局部变量的示例。
        vars.append(SymValueWrapper(``bar'', 99))

        return vars
函数： FrameDecorator.inferior_frame (self): ¶

此方法必须返回底层 gdb.Frame ，此 帧装饰器正在装饰该底层帧。 GDB 在打印帧时，需要底层 帧的内部帧信息来确定如何打印某些 值。
