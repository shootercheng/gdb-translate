4.3 您程序的参数 ¶

您程序的参数可以通过以下命令的参数来指定 run 该命令。它们会传递给一个shell，shell会展开通配符并执行I/O重定向，然后传递给您的程序。您的 SHELL 环境变量（如果存在）指定了使用的shell GDB 如果您没有定义 SHELL, GDB 使用默认的shell（/bin/sh 在Unix系统上）。

在非Unix系统上，程序通常由 GDB, 它通过适当的系统调用来模拟I/O重定向，通配符由程序的启动代码展开，而不是由shell展开。

run 不带参数使用与上一次相同的参数 run, 或者由 set args 命令设置的参数。

set args ¶

指定下次运行程序时要使用的参数。如果 set args 没有参数， run 无参数执行您的程序。一旦您使用参数运行了程序，在下次 set args 之前使用 run 是再次无参数运行程序的唯一方法。

show args ¶

显示程序启动时要给出的参数。
