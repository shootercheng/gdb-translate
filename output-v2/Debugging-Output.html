22.10 关于内部事件的可选消息 ¶

GDB 包含启用来自各种 GDB 子系统的可选调试消息的命令；通常这些命令对 GDB

set exec-done-display

打开或关闭异步命令完成的通知。打开时， GDB 当异步命令完成执行时将打印一条消息。默认值为关闭。

show exec-done-display

显示异步命令完成通知的当前设置。

set debug aarch64 ¶

打开或关闭与ARM AArch64相关的调试消息的显示。默认值为关闭。

show debug aarch64

显示与ARM AArch64相关的调试消息的显示状态。

set debug arch ¶

打开或关闭gdbarch调试信息的显示。默认值为关闭

show debug arch

显示gdbarch调试信息的显示状态。

set debug aix-thread ¶

显示关于AIX线程模块内部工作的调试消息。

show debug aix-thread

显示AIX线程调试信息显示的当前状态。

set debug amd-dbgapi-lib
show debug amd-dbgapi-lib

该 set debug amd-dbgapi-lib log-level level 命令可用于启用来自‘amd - dbgapi’库的诊断消息，其中 级别 可以是：

off

不启用日志记录

error

报告致命错误

warning

报告致命错误和警告

info

报告致命错误、警告和信息消息

verbose

报告所有消息

该 show debug amd-dbgapi-lib log-level 命令显示当前 amd - dbgapi 库的日志级别。

set debug amd-dbgapi
show debug amd-dbgapi

‘set debug amd - dbgapi’命令可用于在‘amd - dbgapi’目标中启用诊断消息。‘show debug amd - dbgapi’命令显示当前设置。请参阅 set debug amd - dbgapi.

set debug check-physname ¶

检查“physname”计算的结果。在读取C的DWARF调试信息时++, GDB 尝试计算每个实体的名称。 GDB 可以根据具体存在的信息以两种不同的方式进行此计算。启用此设置时，会使 GDB 以两种方式计算名称并显示任何差异。

show debug check-physname

显示“physname”检查的当前状态。

set debug coff-pe-read ¶

控制与读取COFF/PE导出符号相关的调试消息的显示。默认值为关闭。

show debug coff-pe-read

显示与读取COFF/PE导出符号相关的调试消息的显示状态。

set debug dwarf-die ¶

读取DWARF DIE后将其转储。该值是要打印的嵌套级别数。值为零将关闭显示。

show debug dwarf-die

显示DWARF DIE调试的当前状态。

set debug dwarf-line ¶

打开或关闭与读取DWARF行表相关的调试消息的显示。默认值为0（关闭）。值为1提供基本信息。大于1的值提供更详细的信息。

show debug dwarf-line

显示DWARF行表调试的当前状态。

set debug dwarf-read ¶

打开或关闭与读取DWARF调试信息相关的调试消息的显示。默认值为0（关闭）。值为1提供基本信息。大于1的值提供更详细的信息。

show debug dwarf-read

显示DWARF读取器调试的当前状态。

set debug displaced ¶

打开或关闭显示 GDB 位移单步执行支持的调试信息。默认值为关闭。

show debug displaced

显示显示 GDB 与位移单步执行相关的调试信息的当前状态。

set debug event ¶

打开或关闭显示 GDB 事件调试信息。默认值为关闭。

show debug event

显示显示 GDB 事件调试 信息的当前状态。

set debug event-loop ¶

控制关于事件循环的调试信息的输出。可能的值为‘关闭’、‘全部’（显示所有调试信息）和‘除UI外全部’（显示除与UI相关事件之外的所有调试信息）。

show debug event-loop

显示关于事件循环的调试信息的显示状态。

set debug expression ¶

打开或关闭关于 GDB 表达式解析的调试信息的显示。默认值为关闭。

show debug expression

显示关于 GDB 表达式解析的调试信息的显示状态。

set debug fbsd-lwp ¶

打开或关闭来自FreeBSD LWP调试支持的调试消息。

show debug fbsd-lwp

显示FreeBSD LWP调试消息的当前状态。

set debug fbsd-nat ¶

打开或关闭来自FreeBSD原生目标的调试消息。

show debug fbsd-nat

显示FreeBSD原生目标调试消息的当前状态。

set debug fortran-array-slicing ¶

打开或关闭显示 GDB Fortran数组切片 调试信息。默认值为关闭。

show debug fortran-array-slicing

显示显示 GDB Fortran数组 切片调试信息的当前状态。

set debug frame ¶

打开或关闭显示 GDB 帧调试信息。默认值为关闭。

show debug frame

显示显示 GDB 帧调试 信息的当前状态。

set debug gnu-nat ¶

打开或关闭来自 GNU/Hurd调试支持的调试消息。

show debug gnu-nat

显示 GNU/Hurd调试消息的当前状态。

set debug infrun ¶

打开或关闭显示 GDB 运行子进程的调试信息。默认值为关闭。 infrun.c 包含GDB的运行时状态机，用于实现诸如单步执行子进程等操作。

show debug infrun

显示 GDB 子进程调试的当前状态。

set debug infcall ¶

打开或关闭显示与 GDB调用的子进程函数调用相关的调试信息。

show debug infcall

显示 GDB 子进程函数调用调试的当前状态。

set debug jit ¶

打开或关闭来自JIT调试支持的调试消息。

show debug jit

显示 GDB JIT调试的当前状态。

set debug linux-nat [on|off] ¶

打开或关闭来自Linux原生目标调试支持的调试消息。

show debug linux-nat

显示Linux原生目标调试消息的当前状态。

set debug linux-namespaces ¶

打开或关闭来自Linux命名空间调试支持的调试消息。

show debug linux-namespaces

显示Linux命名空间调试消息的当前状态。

set debug mach-o ¶

控制与Mach - O符号处理相关的调试消息的显示。默认值为关闭。

show debug mach-o

显示与读取COFF/PE导出符号相关的调试消息的显示状态。

set debug notification ¶

打开或关闭关于远程异步通知的调试消息。默认值为关闭。

show debug notification

显示远程异步通知调试消息的当前状态。

set debug observer ¶

打开或关闭显示 GDB 观察者调试。这包括可观察事件的通知等信息。

show debug observer

显示观察者调试的当前状态。

set debug overload ¶

打开或关闭显示 GDB C++ 重载调试 信息。这包括函数排名等信息。默认值为关闭。

show debug overload

显示显示 GDB C++ 重载 调试信息的当前状态。

set debug parser ¶

打开或关闭表达式解析器调试输出的显示。内部而言，这设置了表达式解析器中的 yydebug 变量。请参阅 跟踪您的解析器 在 Bison中获取详细信息。默认值为关闭。

show debug parser

显示表达式解析器调试的当前状态。

set debug remote ¶

打开或关闭显示通过串行线路往返于远程机器的所有数据包的报告。信息将打印在 GDB 标准输出流上。默认值为关闭。

show debug remote

显示远程数据包显示的状态。

set debug remote-packet-max-chars

当 set debug remote 打开时，设置每个远程数据包显示的最大字符数。这有助于防止 GDB 显示冗长的远程数据包并污染控制台。

默认值为 512, 这意味着 GDB 将在512字节后截断每个远程数据包。

将此选项设置为 unlimited 将禁用截断并输出远程数据包的完整长度。

show debug remote-packet-max-chars

显示用于远程数据包调试的输出字节数。

set debug separate-debug-file

显示单独调试文件搜索调试输出的状态。

show debug serial

set debug solib

show debug solib

显示共享库调试消息的当前状态。

set debug symbol-lookup ¶

打开或关闭与符号查找相关的调试消息的显示。默认值为0（关闭）。值为1提供基本信息。大于1的值提供更详细的信息。

show debug symbol-lookup

显示符号查找调试消息的当前状态。

set debug symfile ¶

打开或关闭与符号文件函数相关的调试消息的显示。默认值为关闭。请参阅 指定文件的命令.

show debug symfile

显示符号文件调试消息的当前状态。

set debug symtab-create ¶

打开或关闭与符号表创建相关的调试消息的显示。默认值为0（关闭）。值为1提供基本信息。大于1的值提供更详细的信息。

show debug symtab-create

显示符号表创建调试的当前状态。

set debug target ¶

打开或关闭显示 GDB 目标调试信息。此信息包括GDB目标级别正在发生的情况。默认值为0。将其设置为1以跟踪事件，设置为2还可跟踪大内存传输的值。

show debug target

显示显示 GDB 目标调试 信息的当前状态。

set debug timestamp ¶

打开或关闭显示带有 GDB 调试信息的时间戳。启用时，每个调试消息前将显示秒和微秒。

show debug timestamp

显示显示带有 GDB 调试信息的时间戳的当前状态。

set debug varobj ¶

打开或关闭显示 GDB 变量对象调试 信息。默认值为关闭。

show debug varobj

显示显示 GDB 变量对象 调试信息的当前状态。

set debug xml ¶

打开或关闭内置XML解析器的调试消息。

show debug xml

显示XML调试消息的当前状态。

set debug breakpoints ¶

打开或关闭显示 GDB 断点插入和移除的调试信息。默认值为关闭。

show debug breakpoints

显示显示 GDB 断点插入和移除的调试信息的当前状态。
