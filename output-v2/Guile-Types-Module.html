23.4.5.2 Guile类型模块 ¶

此模块提供了一组用于处理 <gdb:type> 对象的实用工具。

用法： 

(use-modules (gdb types))
Scheme过程： get-basic-type 类型 ¶

返回 类型 去除了const和volatile限定符， 并将typedef和C++ 引用转换为底层类型。

C++ 示例： 

typedef const int const_int;
const_int foo (3);
const_int& foo_ref (foo);
int main () { return 0; }

然后在gdb中： 

(gdb) start
(gdb) guile (use-modules (gdb) (gdb types))
(gdb) guile (define foo-ref (parse-and-eval "foo_ref"))
(gdb) guile (get-basic-type (value-type foo-ref))
#<gdb:type int>
Scheme过程： type-has-field-deep? 类型 字段 ¶

返回 #t 如果 类型（假定为具有字段的类型， 例如，结构体或联合体）具有字段 字段。 否则返回 #f。 这会搜索基类，而 type-has-field? 不会。

Scheme过程： make-enum-hashtable 枚举类型 ¶

返回一个由 枚举类型生成的Guile哈希表。 哈希表中的元素通过 hashq-ref引用。
