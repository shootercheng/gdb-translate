10.10.3 漂亮打印机命令 ¶

info pretty-printer [object-regexp [name-regexp]] ¶

打印已安装的漂亮打印机列表。 这包括已禁用的漂亮打印机，它们会被标记出来。

对象正则表达式 是一个正则表达式，用于匹配要列出其漂亮打印机的对象。 对象可以是 global, 程序空间的文件 (见 Python 中的程序空间), 以及该程序空间内的目标文件 (见 Python 中的目标文件)。 有关 选择漂亮打印机, 关于如何 GDB 从这三个对象中查找打印机的详细信息。

名称正则表达式 是一个正则表达式，用于匹配要列出的打印机的名称。

disable pretty-printer [object-regexp [name-regexp]] ¶

禁用与 对象正则表达式 和 名称正则表达式匹配的漂亮打印机。 禁用的漂亮打印机不会被遗忘，稍后可以再次启用它。

enable pretty-printer [object-regexp [name-regexp]] ¶

启用与 对象正则表达式 和 名称正则表达式匹配的漂亮打印机。

示例：

假设我们安装了三个漂亮打印机：一个来自 library1.so，名为 foo 它用于打印类型为 foo的对象， 另一个来自 library2.so，名为 bar 它用于打印两种类型的对象， bar1 和 bar2。 

(gdb) info pretty - printer
library1.so:
  foo
library2.so:
  bar
    bar1
    bar2

(gdb) info pretty - printer library2
library2.so:
  bar
    bar1
    bar2

(gdb) disable pretty - printer library1
1 个打印机已禁用
3 个打印机中的 2 个已启用
(gdb) info pretty - printer
library1.so:
  foo [已禁用]
library2.so:
  bar
    bar1
    bar2

(gdb) disable pretty - printer library2 bar;bar1
1 个打印机已禁用
3 个打印机中的 1 个已启用
(gdb) info pretty - printer library2
library2.so:
  bar
    bar1 [已禁用]
    bar2

(gdb) disable pretty - printer library2 bar
1 个打印机已禁用
3 个打印机中的 0 个已启用
(gdb) info pretty - printer
library1.so:
  foo [已禁用]
library2.so:
  bar [已禁用]
    bar1 [已禁用]
    bar2

请注意，对于 bar 整个打印机可以被禁用，每个单独的子打印机也可以被禁用。

默认情况下，使用已启用的漂亮打印机来打印值和帧参数。

打印选项 -raw-values 和 GDB 设置 set print raw-values (见 set print raw - values) 可用于在不应用已启用的漂亮打印机的情况下打印值。

同样，回溯选项 -raw-frame-arguments 和 GDB 设置 set print raw-frame-arguments (见 set print raw - frame - arguments) 可用于在打印帧参数值时忽略已启用的漂亮打印机。
