对于 GDB 查找 x 方法，应使用模块中定义的以下函数注册 x 方法匹配器 gdb.xmethod：

函数： register_xmethod_matcher (locus, matcher, replace=False) ¶

该 matcher 使用 locus进行注册，替换与 matcher 同名的现有匹配器，如果 replace 是 True。 locus 可以是一个 gdb.Objfile 对象（请参阅 Python 中的目标文件），或者是一个 gdb.Progspace 对象（请参阅 Python 中的程序空间），或者 None。如果它是 None，那么 matcher 将全局注册。
