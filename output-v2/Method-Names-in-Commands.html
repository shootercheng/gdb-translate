15.4.4.1 命令中的方法名 ¶

以下命令已扩展为接受 Objective - C 方法名作为行规范：

  • clear
  • break
  • info line
  • jump
  • list

一个完全限定的 Objective - C 方法名的指定方式为 

-[类 方法名]

其中减号用于表示实例方法，加号（未显示）用于表示类方法。类名 类 和方法名 方法名 用方括号括起来，类似于在 Objective - C 源代码中指定消息的方式。例如，要在 create 类的实例方法 Fruit 处设置断点，在当前正在调试的程序中输入： 

break -[Fruit create]

要列出 initialize 类方法周围的十行程序代码，输入： 

list +[NSText initialize]

在当前版本的 GDB, 加号或减号是必需的。在未来版本的 GDB, 加号或减号将是可选的，但你可以用它来缩小搜索范围。也可以只指定一个方法名： 

break create

你必须指定完整的方法名，包括任何冒号。如果你的程序源文件中包含多个 create 方法，你将看到一个实现该方法的类的编号列表。通过编号选择，或者输入 ‘0’ 退出（如果都不适用）。

再举一个例子，要清除在 makeKeyAndOrderFront: 类的 NSWindow 方法处设置的断点，输入： 

clear -[NSWindow makeKeyAndOrderFront:]
