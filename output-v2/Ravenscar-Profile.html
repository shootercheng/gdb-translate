15.4.10.9 使用 Ravenscar 配置文件时的任务支持 ¶

Ravenscar 配置文件 是 Ada 任务特性的一个子集，专门为有安全关键实时需求的系统设计。

set ravenscar task-switching on ¶

在调试使用 Ravenscar 配置文件的程序时允许任务切换。这是默认设置。

set ravenscar task-switching off ¶

在调试使用 Ravenscar 配置文件的程序时关闭任务切换。这主要是为了禁用添加对 Ravenscar 配置文件支持的代码，以防任一代码中的错误 GDB 或 Ravenscar 运行时的错误导致 GDB 无法正常工作。为了生效，此命令应在程序启动前运行。

show ravenscar task-switching ¶

显示在使用 Ravenscar 配置文件的程序中是否可以在任务之间切换。

当启用 Ravenscar 任务切换时，Ravenscar 任务会被 GDB 宣布，就好像它们是线程一样： 

(gdb) continue
[新 Ravenscar 线程 0x2b8f0]

Ravenscar 任务和底层 CPU 线程都会显示在 info threads的输出中： 

(gdb) info threads
  Id   目标 ID                  帧
  1    线程 1 (CPU#0 [运行中]) simple () 在 simple.adb:10
  2    线程 2 (CPU#1 [运行中]) 0x0000000000003d34 在 __gnat_initialize_cpu_devices ()
  3    线程 3 (CPU#2 [运行中]) 0x0000000000003d28 在 __gnat_initialize_cpu_devices ()
  4    线程 4 (CPU#3 [暂停 ]) 0x000000000000c6ec 在 system.task_primitives.operations.idle ()
* 5    Ravenscar 线程 0x2b8f0   simple () 在 simple.adb:10
  6    Ravenscar 线程 0x2f150   0x000000000000c6ec 在 system.task_primitives.operations.idle ()

在 GDB 中对 Ravenscar 支持的一个已知限制是，目前无法单步执行运行时初始化序列。如果需要调试此代码，应使用 set ravenscar task-switching off.
