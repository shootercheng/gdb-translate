15.4.3 Go ¶

GDB 可用于调试用Go编写并使用以下编译器编译的程序 gccgo 或 6g 编译器。

以下是Go特定功能和限制的总结：

The current Go package ¶

在指定全局变量和函数时，不需要指定当前包的名称。

例如，给定程序： 

package main
var myglob = "Shall we?"
func main () {
  // ...
}

当在内部停止时 main 以下两种方式都可行： 

(gdb) p myglob
(gdb) p main.myglob
Builtin Go types ¶

该 string 类型被 GDB 识别并作为字符串打印。

Builtin Go functions ¶

该 GDB 表达式解析器识别 unsafe.Sizeof 函数并在内部处理它。

Restrictions on Go expressions ¶

除了 &^之外，所有Go运算符都受支持。 Go _ “空白标识符”不受支持。 不支持指针的自动解引用。
