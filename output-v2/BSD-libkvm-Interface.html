21.1.1 BSD libkvm 接口 ¶

基于 BSD 的系统（FreeBSD/NetBSD/OpenBSD）有一个内核内存接口，它为访问内核虚拟内存映像提供了统一的接口，包括实时系统和崩溃转储。 GDB 使用此接口允许你在许多原生 BSD 配置上调试实时内核和内核崩溃转储。这被实现为一个特殊的 kvm 调试目标。对于调试实时系统，将当前正在运行的内核加载到 GDB 并连接到 kvm 目标： 

(gdb) target kvm

对于调试崩溃转储，提供崩溃转储的文件名作为参数： 

(gdb) target kvm /var/crash/bsd.0

一旦连接到 kvm 目标，以下命令可用：

kvm pcb ¶

从 进程控制块 (PCB) 地址设置当前上下文。

kvm proc

从进程地址设置当前上下文。此命令在现代 FreeBSD 系统上不可用。
