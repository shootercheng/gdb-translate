15.4.10.10 Ada 源字符集 ¶

GNAT 编译器支持源文件使用多种字符集。请参阅 字符集控制. GDB 也支持此功能。

set ada source-charset charset ¶

为 Ada 设置源字符集。该字符集必须为 GNAT 所支持。由于此设置会影响对程序调试信息中符号的解码，因此应尽早进行设置。默认值为 ISO-8859-1, 因为这也是 GNAT 的默认设置。

show ada source-charset ¶

显示当前 Ada 的源字符集。
