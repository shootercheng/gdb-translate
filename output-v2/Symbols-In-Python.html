23.3.2.30 符号的 Python 表示形式 ¶

GDB 将每个变量、函数和类型表示为符号表中的一个条目。请参阅 检查符号表.同样，Python 在 GDB 使用 gdb.Symbol 对象。

以下与符号相关的函数在 gdb 模块：

函数： gdb.lookup_symbol (名称 , 块 ]]) ¶

此函数按名称搜索符号。搜索范围可以限制在可选的域和块参数中定义的参数。

名称 是符号的名称。它必须是一个字符串。可选的 块 参数将搜索限制在该 块中可见的符号。 块 参数必须是一个 gdb.Block 对象。如果省略，则使用当前帧的块。可选的 域 参数将搜索限制在域类型。 域 参数必须是在 gdb 模块中定义并在本章后面描述的域常量。

结果是一个包含两个元素的元组。第一个元素是一个 gdb.Symbol 对象，或者如果未找到符号则为 None 。如果找到符号，第二个元素是 True 如果符号是方法对象的字段（例如， this 在 C 中++），否则为 False。如果未找到符号，第二个元素是 False。

函数： gdb.lookup_global_symbol (名称 [, 域]) ¶

此函数按名称搜索全局符号。搜索范围可以通过域参数进行限制。

名称 是符号的名称。它必须是一个字符串。可选的 域 参数将搜索限制在域类型。 域 参数必须是在 gdb 模块中定义并在本章后面描述的域常量。

结果是一个 gdb.Symbol 对象，或者如果未找到符号则为 None 。

函数： gdb.lookup_static_symbol (名称 [, 域]) ¶

此函数按名称搜索具有静态链接的全局符号。搜索范围可以通过域参数进行限制。

名称 是符号的名称。它必须是一个字符串。可选的 域 参数将搜索限制在域类型。 域 参数必须是在 gdb 模块中定义并在本章后面描述的域常量。

结果是一个 gdb.Symbol 对象，或者如果未找到符号则为 None 。

请注意，此函数不会找到函数作用域内的静态变量。要查找此类变量，请遍历函数的 gdb.Block 并检查 block.addr_class 是否为 gdb.SYMBOL_LOC_STATIC.

可能存在多个具有相同名称的具有静态链接的全局符号。此函数将只返回它找到的第一个匹配符号。找到哪个符号取决于 GDB 当前停止的位置，因为 GDB 将首先在当前目标文件中搜索匹配的符号，然后搜索所有其他目标文件。如果应用程序尚未运行，则 GDB 将按调试信息中出现的顺序搜索所有目标文件。

函数： gdb.lookup_static_symbols (名称 [, 域]) ¶

与……类似 gdb.lookup_static_symbol, 此函数按名称搜索具有静态链接的全局符号，并可选择通过域参数进行限制。 但是，此函数返回找到的所有匹配符号的列表，而不仅仅是第一个。

名称 是符号的名称。 它必须是一个字符串。可选的 域 参数将搜索限制在域类型。 域 参数必须是在 gdb 模块中定义并在本章后面描述的域常量。

结果是一个 gdb.Symbol 对象列表，如果没有找到匹配的符号，该列表可能为空。

请注意，此函数不会找到函数作用域内的静态变量。要查找此类变量，请遍历函数的 gdb.Block 并检查 block.addr_class 是否为 gdb.SYMBOL_LOC_STATIC。

一个 gdb.Symbol 对象具有以下属性：

变量： Symbol.type ¶

符号的类型，或者 None 如果没有记录类型。此属性表示为一个 gdb.Type 对象。请参阅 Python 中的类型。此属性不可写。

变量： Symbol.symtab ¶

符号出现的符号表。此属性表示为一个 gdb.Symtab 对象。请参阅 Python 中的符号表表示形式。此属性不可写。

变量： Symbol.line ¶

符号在源代码中定义的行号。这是一个整数。

变量： Symbol.name ¶

符号的名称，以字符串形式表示。此属性不可写。

变量： Symbol.linkage_name ¶

链接器使用的符号名称（即，可能是经过修饰的）。此属性不可写。

变量： Symbol.print_name ¶

适合输出的符号名称形式。这可以是 name 或 linkage_name, 具体取决于用户是否要求 GDB 显示去修饰或修饰后的名称。

变量： Symbol.addr_class ¶

符号的地址类。这对如何查找符号的值进行分类。每个地址类是在 gdb 模块中定义并在本章后面描述的常量。

变量： Symbol.domain ¶

符号的域。每个域是在 gdb 模块中定义并在本章后面描述的常量。

变量： Symbol.needs_frame ¶

如果计算此符号的值需要一个帧（请参阅 True 从 Python 访问 inferior 栈帧 ），则为，否则为 False 。通常，局部变量需要一个帧，但其他符号不需要。

变量： Symbol.is_argument ¶

True 如果符号是函数的参数。

变量： Symbol.is_artificial ¶

True 如果符号是人工的。人工符号是编译器引入的符号。

变量： Symbol.is_constant ¶

True 如果符号是常量。

变量： Symbol.is_function ¶

True 如果符号是函数或方法。

变量： Symbol.is_variable ¶

True 如果符号是变量，而不是函数或类型之类的东西。请注意，对于参数，此属性也返回 False 。

一个 gdb.Symbol 对象具有以下方法：

函数： Symbol.is_valid () ¶

如果 True 对象有效，则返回 gdb.Symbol ，否则返回 False 。如果 gdb.Symbol 对象所引用的符号在 GDB 中不再存在，则该对象可能会变得无效。如果在调用方法时对象无效，所有其他 gdb.Symbol 方法将抛出异常。

函数： Symbol.value ([帧]) ¶

计算符号的值，作为一个 gdb.Value。对于函数，这计算函数的地址，并转换为适当的类型。如果符号需要一个帧来计算其值，则必须提供 帧 。如果没有提供 帧 ，或者 帧 无效，则此方法将抛出异常。

在 gdb.Symbol 中可用的域类别在 gdb 模块中表示为常量：

gdb.SYMBOL_UNDEF_DOMAIN ¶

当未发现域或以下域都不适用时使用。这通常表示符号信息或 GDB对符号的处理存在错误。

gdb.SYMBOL_VAR_DOMAIN ¶

此域包含变量。

gdb.SYMBOL_FUNCTION_DOMAIN ¶

此域包含函数。

gdb.SYMBOL_TYPE_DOMAIN ¶

此域包含类型。在类似 C 的语言中，使用标签（出现在 struct, 或 union关键字之后的名称）的类型不会出现在这里；在其他语言中，所有类型都在此域中。 enum

gdb.SYMBOL_STRUCT_DOMAIN ¶

此域保存结构体、联合体和枚举标签名称。此域仅用于类似 C 的语言。例如，在以下代码中： 

struct type_one { int x; };
typedef struct type_one type_two;

这里 type_one 将在 SYMBOL_STRUCT_DOMAIN, 但 type_two 将在 SYMBOL_TYPE_DOMAIN.

gdb.SYMBOL_LABEL_DOMAIN ¶

此域包含标签（用于 goto）的名称。

gdb.SYMBOL_MODULE_DOMAIN ¶

此域包含 Fortran 模块类型的名称。

gdb.SYMBOL_COMMON_BLOCK_DOMAIN ¶

此域包含 Fortran 公共块的名称。

在搜索符号时，可以将所需的域常量直接传递给查找函数。例如： 

symbol = gdb.lookup_symbol ("name", domain=gdb.SYMBOL_VAR_DOMAIN)

对于更复杂的搜索，有一组相应的常量，每个常量的名称都基于前面的常量，但用SEARCH前缀替换SYMBOL前缀；例如， SEARCH_LABEL_DOMAIN. 这些常量可以通过按位或组合形成一个搜索常量，例如： 

symbol = gdb.lookup_symbol ("name",
                            domain=gdb.SEARCH_VAR_DOMAIN | gdb.SEARCH_TYPE_DOMAIN)

在 gdb.Symbol 中可用的地址类类别在 gdb 模块中表示为常量：

gdb.SYMBOL_LOC_UNDEF ¶

如果地址类返回此值，则表示符号信息或 GDB对符号的处理存在错误。

gdb.SYMBOL_LOC_CONST ¶

值是常量整数。

gdb.SYMBOL_LOC_STATIC ¶

值位于固定地址。

gdb.SYMBOL_LOC_REGISTER ¶

值在寄存器中。

gdb.SYMBOL_LOC_ARG ¶

值是一个参数。此值位于帧的参数列表中符号内部存储的偏移量处。

gdb.SYMBOL_LOC_REF_ARG ¶

值的地址存储在帧的参数列表中。与 LOC_ARG 类似，只是值的地址存储在偏移量处，而不是值本身。

gdb.SYMBOL_LOC_REGPARM_ADDR ¶

值是指定的寄存器。与 LOC_REGISTER 类似，只是寄存器保存的是参数的地址，而不是参数本身。

gdb.SYMBOL_LOC_LOCAL ¶

值是局部变量。

gdb.SYMBOL_LOC_TYPEDEF ¶

值未使用。 SYMBOL_STRUCT_DOMAIN 域中的符号都具有此类别。

gdb.SYMBOL_LOC_LABEL ¶

值是标签。

gdb.SYMBOL_LOC_BLOCK ¶

值是块。

gdb.SYMBOL_LOC_CONST_BYTES ¶

值是字节序列。

gdb.SYMBOL_LOC_UNRESOLVED ¶

值位于固定地址，但每次引用变量时，变量的地址必须从最小符号表中确定。

gdb.SYMBOL_LOC_OPTIMIZED_OUT ¶

该值实际上在程序中不存在。

gdb.SYMBOL_LOC_COMPUTED ¶

值的地址是一个计算位置。

gdb.SYMBOL_LOC_COMMON_BLOCK ¶

值的地址是一个符号。这仅用于 Fortran 公共块。
