23.6 多种扩展语言 ¶

Guile 和 Python 扩展语言不共享任何状态，并且通常不会相互干扰。不过，有一些事情需要注意。

  • Python 优先

23.6.1 Python 优先 ¶

Python 是 GDB的第一种扩展语言，为了避免破坏现有行为，Python 优先。这通常通过“先到先得”原则来解决。 GDB 维护一个已启用的扩展语言列表，当它调用扩展语言时（例如，对一个值进行漂亮打印），它会依次尝试每种语言，直到某个扩展语言表明它已经完成了请求（例如，返回了一个值的漂亮打印形式）。这也适用于执行此类请求时出现的错误：例如，如果在尝试对一个对象进行漂亮打印时发生错误，则会报告该错误，并且不会尝试后续的扩展语言。
