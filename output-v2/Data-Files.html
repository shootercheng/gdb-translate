18.8 GDB数据文件 ¶

GDB 有时会读取一个辅助数据文件。这些文件存放在一个被称为 数据目录的目录中。

你可以设置数据目录的名称，并查看当前正在使用的名称 GDB 。

set data-directory directory ¶

设置 GDB 搜索辅助数据文件的目录为 目录。

show data-directory ¶

显示 GDB 搜索辅助数据文件的目录。

你可以在配置时使用‘--with-gdb-datadir’选项来设置默认的数据目录。如果数据目录位于 GDB的已配置二进制前缀（使用‘--prefix’或 ‘--exec-prefix’设置）内，那么如果已安装的 GDB 被移动到新位置，默认的数据目录将自动更新。

数据目录也可以通过 --data-directory 命令行选项指定。 请参阅 选择模式。
