33.2.2 Readline 移动命令 ¶

上述表格描述了为了编辑输入行所需的最基本按键操作。为了方便起见，除了 Ctrl + b, Ctrl + f, Ctrl + d和 删除键之外，还添加了许多其他命令。以下是一些用于在输入行上更快速移动的命令。

Ctrl + a

移动到行首。

Ctrl + e

移动到行尾。

Alt + f

向前移动一个单词，这里的单词由字母和数字组成。

Alt + b

向后移动一个单词。

Ctrl + l

清屏，并将当前行重新打印在屏幕顶部。

注意 Ctrl + f 是向前移动一个字符，而 Alt + f 是向前移动一个单词。通常的约定是，控制键操作针对字符，而元键操作针对单词。
