附录 B 格式化文档 ¶

该 GDB 4 版本包含一个已格式化的参考卡，可使用 PostScript 或 Ghostscript 进行打印，位于 gdb 主源目录的子目录中20. 如果你可以在打印机上使用 PostScript 或 Ghostscript，你可以立即使用 refcard.ps.

该版本还包含参考卡的源文件。你可以使用 TeX 对其进行格式化，方法是输入： 

make refcard.dvi

该 GDB 参考卡设计为在美国“信纸”尺寸的纸张上以 横向 模式打印；也就是说，在一张宽 11 英寸、高 8.5 英寸的纸张上。你需要将这种打印形式作为选项指定给你的 DVI 输出程序。

所有关于 GDB 的文档都是机器可读发行版的一部分。文档采用 Texinfo 格式编写，这是一种使用单个源文件同时生成在线信息和印刷手册的文档系统。你可以使用 Info 格式化命令之一来创建文档的在线版本，使用 TeX（或 texi2roff) 来排版印刷版本。

GDB 在 gdb 子目录中包含了本手册在线 Info 版本的已格式化副本。主 Info 文件是 gdb-17.2/gdb/gdb.info, 它引用了同一目录中匹配‘gdb.info*’的从属文件。如有必要，你可以打印这些文件，或使用任何编辑器阅读它们；但使用 info 子系统在 GNU Emacs 或独立的 info 程序中阅读会更容易，该程序是 GNU Texinfo 发行版的一部分。

如果你想自己格式化这些 Info 文件，你需要一个 Info 格式化程序，例如 texinfo-format-buffer 或 makeinfo.

如果你已经 makeinfo 安装，并且位于顶级 GDB 源目录（gdb-17.2, 对于 17.2 版本而言），你可以通过输入以下命令来生成 Info 文件： 

cd gdb
make gdb.info

如果你想排版并打印本手册的副本，你需要 TeX、一个用于打印其 DVI 输出文件的程序，以及 texinfo.tex, 即 Texinfo 定义文件。

TeX 是一个排版程序；它不直接打印文件，而是生成称为 DVI 文件的输出文件。要打印排版好的文档，你需要一个程序来打印 DVI 文件。如果你的系统安装了 TeX，很可能也安装了这样的程序。具体使用的命令取决于你的系统； lpr -d 很常见；另一个（用于 PostScript 设备）是 dvips. 该 DVI 打印命令可能需要一个不带任何扩展名或带有‘.dvi’扩展名的文件名。

TeX 还需要一个名为 texinfo.tex. 的宏定义文件。该文件告诉 TeX 如何排版以 Texinfo 格式编写的文档。TeX 本身既不能读取也不能排版 Texinfo 文件。 texinfo.tex 随 GDB 一起分发，位于 gdb-版本号/texinfo 目录中。

如果你安装了 TeX 和一个 DVI 打印机程序，你可以排版并打印本手册。首先切换到 gdb 主源目录的子目录（例如，切换到 gdb-17.2/gdb) 并输入： 

make gdb.dvi

然后将 gdb.dvi 交给你的 DVI 打印程序。
