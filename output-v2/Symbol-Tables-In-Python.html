23.3.2.31 Python 中的符号表表示 ¶

对由以下工具维护的符号表数据的访问 GDB 在目标程序上 通过两个对象向 Python 公开： gdb.Symtab_and_line 和 gdb.Symtab。 帧的符号表和行数据从 find_sal 方法在 gdb.Frame 对象中返回。 请参阅 从 Python 访问目标程序的栈帧.

有关 GDB的符号表管理的更多信息，请参阅 检查符号表, 以获取更多信息。

一个 gdb.Symtab_and_line 对象具有以下属性：

变量： Symtab_and_line.symtab ¶

此帧的符号表对象（gdb.Symtab）。 此属性不可写。

变量： Symtab_and_line.pc ¶

指示当前源行代码所占用地址范围的起始位置。 此属性不可写。

变量： Symtab_and_line.last ¶

指示当前源行代码所占用地址范围的结束位置。 此属性不可写。

变量： Symtab_and_line.line ¶

指示此对象的当前行号。 此属性不可写。

一个 gdb.Symtab_and_line 对象具有以下方法：

函数： Symtab_and_line.is_valid () ¶

如果 True 对象有效，则返回 gdb.Symtab_and_line ， False 如果无效。 一个 gdb.Symtab_and_line 对象如果它所引用的符号表和行对象在 GDB 中不再存在，则可能变为无效。 如果在调用方法时对象无效，所有其他 gdb.Symtab_and_line 方法将抛出异常。

一个 gdb.Symtab 对象具有以下属性：

变量： Symtab.filename ¶

符号表的源文件名。 此属性不可写。

变量： Symtab.objfile ¶

符号表的底层目标文件。 请参阅 Python 中的目标文件. 此属性不可写。

变量： Symtab.producer ¶

编译符号表中代码的程序的名称和可能的版本号。 此字符串的内容由编译器决定。 如果没有可用的生产者信息，则返回 None 。 此属性不可写。

一个 gdb.Symtab 对象具有以下方法：

函数： Symtab.is_valid () ¶

如果 True 对象有效，则返回 gdb.Symtab ， False 如果无效。 一个 gdb.Symtab 对象如果它所引用的符号表在 GDB 中不再存在，则可能变为无效。 如果在调用方法时对象无效，所有其他 gdb.Symtab 方法将抛出异常。

函数： Symtab.fullname () ¶

返回符号表的源绝对文件名。

函数： Symtab.global_block () ¶

返回底层符号表的全局块。 请参阅 从 Python 访问块.

函数： Symtab.static_block () ¶

返回底层符号表的静态块。 请参阅 从 Python 访问块.

函数： Symtab.linetable () ¶

返回与符号表关联的行表。 请参阅 使用 Python 操作行表.
