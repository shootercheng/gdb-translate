E.18 线程列表格式 ¶

为了高效更新线程列表及其属性， GDB 会发送 ‘qXfer:threads:read’ 数据包 （参见 qXfer 线程读取) 并获取具有以下结构的 XML 文档： 

<?xml version="1.0"?>
<threads>
    <thread id="id" core="0" name="name" id_str="线程 12.34" handle="1a2b3c">
    ... 描述 ...
    </thread>
</threads>

每个 ‘线程’ 元素必须有 ‘id’ 属性，用于标识线程（参见 线程 ID 语法)。 ‘核心’ 属性（如果存在）指定线程最后执行所在的处理器核心。 ‘名称’ 属性（如果存在）指定线程的可读名称。 ‘线程’ 元素的内容被解释为可读的辅助信息。 ‘id_str’ 属性（如果存在）指定 GDB 应作为线程的目标 ID 打印的内容（例如在 ‘info threads’ 命令中或切换到该线程时）。 ‘句柄’ 属性（如果存在）是线程句柄的十六进制编码表示。
