33.1 行编辑简介 ¶

以下段落描述了用于表示按键操作的符号表示法。

文本 C - k 读作‘Control - K’，表示在按下Control键的同时按下 k 键所产生的字符。

文本 M - k 读作‘Meta - K’，表示在按下Meta键（如果有的话）的同时按下 k 键所产生的字符。 Meta键在许多键盘上标为 ALT 。 在有两个标有 ALT 键（通常在空格键两侧）的键盘上，左侧的 ALT 键通常被设置为作为Meta键使用。 右侧的 ALT 键也可以配置为作为Meta键使用，或者配置为其他修饰键，例如用于输入带重音字符的组合键。

如果您没有Meta键或 ALT 键，或者没有其他键作为Meta键使用，可以通过先输入 ESC ，然后输入 k来生成相同的按键操作。 这两种操作过程都称为 元化 k 键。

文本 M - C - k 读作‘Meta - Control - k’，表示对 元化 C - k所产生的字符。

此外，有几个键有它们自己的名称。具体来说， DEL, ESC, LFD, SPC, RET, 以及 TAB 在本文或初始化文件（参见 Readline初始化文件）中出现时都代表它们自身。 如果您的键盘没有 LFD 键，输入 C - j 将产生所需的字符。 RET 键在某些键盘上可能标为 Return 或 Enter 。
