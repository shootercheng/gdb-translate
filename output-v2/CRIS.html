21.3.10 CRIS ¶

当配置为调试 CRIS 时， GDB 提供以下特定于 CRIS 的命令：

set cris-version ver ¶

将当前 CRIS 版本设置为 版本, 可以是 ‘10’ 或 ‘32’。 CRIS 版本会影响寄存器名称和大小。 此命令在 CRIS 版本自动检测失败的情况下很有用。

show cris-version

显示当前 CRIS 版本。

set cris-dwarf2-cfi ¶

设置在 CRIS 调试中使用 DWARF - 2 CFI。 默认值为 ‘开启’。 当使用关闭’ 时，将其更改为 ‘ gcc-cris 其版本低于 R59的情况。

show cris-dwarf2-cfi

显示当前使用 DWARF - 2 CFI 的状态。

set cris-mode mode ¶

将当前 CRIS 模式设置为 模式. 只有在专家模式下调试时才应更改此设置，在这种情况下，应将其设置为 ‘专家’（默认值为 ‘正常’）。

show cris-mode

显示当前 CRIS 模式。
