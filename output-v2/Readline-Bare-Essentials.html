33.2.1 Readline基础要点 ¶

要在一行中输入字符，只需直接输入即可。输入的字符会出现在光标所在位置，然后光标向右移动一格。如果输入错误，可以使用退格字符来回退并删除输错的字符。

有时你可能输错了一个字符，直到输入了其他几个字符后才发现错误。在这种情况下，你可以输入 Ctrl + b 将光标向左移动，然后纠正错误。之后，你可以使用 Ctrl + f将光标向右移动。

当你在一行中间添加文本时，你会注意到光标右侧的字符会“被挤开”，为你插入的文本腾出空间。同样，当你删除光标后面的文本时，光标右侧的字符会“拉回”以填补因删除文本而产生的空白。以下是编辑输入行文本的基础要点列表。

Ctrl + b

向左移动一个字符。

Ctrl + f

向右移动一个字符。

删除键 或 退格键

删除光标左侧的字符。

Ctrl + d

删除光标所在位置的字符。

打印字符

在光标处将字符插入到行中。

Ctrl + _ 或 Ctrl + x Ctrl + u

撤销上一个编辑命令。你可以一直撤销到空行。

（根据你的配置， 退格键 可能被设置为删除光标左侧的字符，而 删除键 被设置为删除光标所在位置的字符，就像 Ctrl + d一样，而不是删除光标左侧的字符。）
