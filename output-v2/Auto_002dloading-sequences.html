23.1.5 控制自动加载原生 GDB 脚本 ¶

当读取一个新的目标文件时（例如，由于 file 命令，或者因为被调试程序加载了一个共享库）， GDB 会查找命令文件 目标文件-gdb.gdb. 请参阅 自动加载扩展.

可以启用或禁用自动加载， 并且可以打印自动加载的脚本列表。

set auto-load gdb-scripts [on|off] ¶

启用或禁用命令脚本固定序列的自动加载。

show auto-load gdb-scripts ¶

显示命令脚本固定序列的自动加载是启用还是 禁用。

info auto-load gdb-scripts [regexp] ¶

打印所有已 GDB 自动加载的命令脚本固定序列的列表。

如果 正则表达式 被提供，则只打印名称匹配的命令脚本固定序列。
