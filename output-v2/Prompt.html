22.1 提示符 ¶

GDB 通过打印一个称为 提示符的字符串来表明它已准备好读取命令。这个字符串通常是 ‘(gdb)’。你可以使用 set prompt 命令来更改提示符字符串。例如，在调试 GDB 时，在其中一个 GDB会话中更改提示符很有用，这样你就可以始终知道你正在与哪个会话交互。 GDB

注意： set prompt 不会在你设置的提示符后自动添加空格。这允许你设置以空格结尾的提示符或不以空格结尾的提示符。

set prompt newprompt ¶

指示 GDB 从此使用 newprompt 作为其提示符字符串。

例如，这将设置一个蓝色的 “(gdb)” 提示符： 

set prompt \001\033[0;34m\002(gdb)\001\033[0m\002

它使用 ‘\001’ 和 ‘\002’ 来开始和结束非打印字符序列，以确保它们不被计入字符串长度。

show prompt ¶

打印一行形式为：‘Gdb的提示符是： your-prompt’

启用Python脚本支持的 GDB 版本有提示符扩展。与这些扩展交互的命令如下：

set extended-prompt prompt ¶

设置一个允许替换的扩展提示符。请参阅 gdb.prompt, 了解可用于替换的转义序列列表。每次显示提示符时，作为提示符字符串一部分指定的任何转义序列都会被相应的字符串替换。

例如： 

set extended-prompt 当前工作目录: \w (gdb)

请注意，当设置了扩展提示符时，它会控制 prompt_hook 钩子。有关更多信息，请参阅 prompt_hook。

show extended-prompt ¶

打印扩展提示符。每次显示提示符时，使用 set extended-prompt, 作为提示符字符串一部分指定的任何转义序列都会被相应的字符串替换。
