E.14.10 文件输入/输出示例 ¶

写调用的示例序列，文件描述符为 3，缓冲区位于目标地址 0x1234，应写入 6 个字节： 

<- Fwrite,3,1234,6
请求从目标读取内存
-> m1234,6
<- XXXXXX
返回 "已写入 6 个字节"
-> F6

读调用的示例序列，文件描述符为 3，缓冲区位于目标地址 0x1234，应读取 6 个字节： 

<- Fread,3,1234,6
请求向目标写入内存
-> X1234,6:XXXXXX
返回 "已读取 6 个字节"
-> F6

读调用的示例序列，由于无效的文件描述符，调用在主机上失败（EBADF): 

<- Fread,3,1234,6
-> F-1,9

读调用的示例序列，用户在主机上的系统调用被调用之前按下 Ctrl-c ： 

<- Fread,3,1234,6
-> F-1,4,C
<- T02

读调用的示例序列，用户在主机上的系统调用被调用之后按下 Ctrl-c ： 

<- Fread,3,1234,6
-> X1234,6:XXXXXX
<- T02
