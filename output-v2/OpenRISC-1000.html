21.3.7 OpenRISC 1000 ¶

OpenRISC 1000 提供了一种免费的 RISC 指令集架构。它主要作为软核提供，可以在赛灵思（Xilinx）、阿尔特拉（Altera）和其他现场可编程门阵列（FPGA）上运行。

GDB 对于 OpenRISC，在连接到目标时支持以下命令：

target sim ¶

运行内置的 CPU 模拟器，该模拟器可以运行非常基本的程序，但不支持大多数硬件功能，如内存管理单元（MMU）。对于更复杂的用例，建议用户运行外部目标，并使用‘目标远程连接’进行连接。

示例： target sim

set debug or1k

切换是否显示来自 OpenRISC 目标支持子系统的特定于 OpenRISC 的调试消息。

show debug or1k

显示是否启用了特定于 OpenRISC 的调试消息。
