10.10.2 漂亮打印机示例 ¶

以下是一个 C++ std::string 在没有漂亮打印机时的样子： 

(gdb) print s
$1 = {
  static npos = 4294967295, 
  _M_dataplus = {
    <std::allocator<char>> = {
      <__gnu_cxx::new_allocator<char>> = {
        <无数据字段>}, <无数据字段>
      },
    std::basic_string<char, std::char_traits<char>,
      std::allocator<char> >::_Alloc_hider 的成员:
    _M_p = 0x804a014 "abcd"
  }
}

使用针对 std::string 的漂亮打印机时，仅打印内容： 

(gdb) print s
$2 = "abcd"
