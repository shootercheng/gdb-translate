23.3 扩展 GDB 使用 Python ¶

您可以扩展 GDB 使用 Python 编程语言. 此功能仅在 GDB 配置时使用了 --with-python时可用。

GDB 使用的 Python 脚本 GDB 应安装在 数据目录/python中，其中 数据目录 是在启动时确定的数据目录（请参阅 GDB GDB 数据文件）。 此目录，即 Python 目录，会自动添加到 Python 搜索路径中，以便 Python 解释器能够找到安装在此位置的所有脚本。

此外， GDB 用 Python 编写且位于 数据目录/python/gdb/command 或 数据目录/python/gdb/function 目录中的命令和便利函数会在 GDB GDB 启动时自动导入。
