23.3.2.36 颜色的 Python 表示形式 ¶

您可以将 gdb.Color 的实例赋值给使用 value 。 Parameter 实例的 PARAM_COLOR创建的

gdb.Color 可以引用调色板中的索引，或者包含某个颜色空间中的颜色分量。

函数： Color.__init__ ([值 [, 颜色空间]]) ¶

值 可以是 None （表示终端的默认颜色）、调色板中颜色的整数索引、包含颜色分量的元组或字符串。字符串可以是 ‘#RRGGBB’ 格式的十六进制 RGB 三元组，或者是以下颜色名称之一： ‘无’（表示终端的默认颜色）、‘黑色’、‘红色’、 ‘绿色’、‘黄色’、‘蓝色’、‘品红色’、‘青色’、 或 ‘白色’。

颜色空间 应该是下面列出的 ‘COLORSPACE_’ 常量之一。此参数告诉 GDB 该 值 所属的颜色空间。

变量： Color.is_none ¶

此属性为布尔值。如果其值为 True ，则颜色为终端的默认颜色。

变量： Color.is_indexed ¶

此属性为布尔值。如果其值为 True ，则颜色是索引颜色，即属于某个调色板。

变量： Color.is_direct ¶

此属性为布尔值。如果其值为 True ，则此对象描述的是 ISO/IEC 8613 - 6 意义上的直接颜色。

变量： Color.index ¶

如果 is_indexed 为 True，则此属性存在。其整数值是调色板中颜色的索引。

变量： Color.components ¶

如果 is_direct 为 True，则此属性存在。其值是包含颜色整数分量的元组。

函数： Color.escape_sequence (是否为前景色) ¶

返回将终端颜色更改为该颜色的字符串。

如果 是否为前景色 为 True, 则返回的序列将更改前景色。否则，返回的序列将更改背景色。

如果当前样式已禁用（请参阅 设置样式启用），则此方法将返回空字符串。

初始化颜色时，必须指定其颜色空间。可用的颜色空间由以下模块中定义的常量表示 gdb 模块：

gdb.COLORSPACE_MONOCHROME ¶

仅包含终端默认颜色的调色板。

gdb.COLORSPACE_ANSI_8COLOR ¶

包含 ISO/IEC 6429 标准的八种标准颜色（“黑色”、“红色”、“绿色”等）的调色板。

gdb.COLORSPACE_AIXTERM_16COLOR ¶

包含 16 种颜色的调色板。前八种是 ISO/IEC 6429 的标准颜色（“黑色”、“红色”、“绿色”等）。接下来的八种是它们的亮色版本。

gdb.COLORSPACE_XTERM_256COLOR ¶

包含 256 种颜色的调色板。前 16 种来自 COLORSPACE_AIXTERM_16COLOR。接下来的 216 种颜色是 6x6x6 的 RGB 立方体。最后 24 种颜色构成灰度渐变。

gdb.COLORSPACE_RGB_24BIT ¶

直接的 24 位 RGB 颜色。

无法对 Color 类进行子类化。
