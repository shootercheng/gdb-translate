21.3.6 MIPS 嵌入式 ¶

GDB 支持这些特殊命令用于 MIPS 目标板：

set mipsfpu double ¶
set mipsfpu single
set mipsfpu none
set mipsfpu auto
show mipsfpu

如果你的目标板不支持 MIPS 浮点 协处理器，你应该使用命令 ‘set mipsfpu none’（如果你 需要这样做，你可能希望将该命令放在你的 GDB 初始化 文件中）。这会告诉 GDB 如何找到返回浮点值的函数的返回值。它还允许 GDB 在目标板上调用函数时避免保存浮点寄存器。如果你使用的是仅支持单精度浮点的浮点协处理器，就像在 R4650 处理器上一样，使用命令 ‘set mipsfpu single’。默认的 双精度浮点协处理器可以使用 ‘set mipsfpu double’ 来选择。

在以前的版本中，只有双精度或无 浮点这两种选择，所以 ‘set mipsfpu on’ 将选择双精度 而 ‘set mipsfpu off’ 将选择无浮点。
