gstack ¶

gstack [-h | –帮助] [-v | –版本] 进程 ID

打印具有进程 ID 的正在运行程序的堆栈跟踪信息 进程 ID. 如果该进程 是多线程的， gstack 则输出该进程中存在的每个线程的回溯信息。

该脚本会调用 GDB, 附加到给定的进程 ID，打印堆栈跟踪信息， 然后从该进程分离。

gstack 如果 gdb 由于任何原因（例如进程 ID 不存在或 没有足够的权限附加到该进程）无法附加到给定的进程 ID，则以非零状态退出。

--help
-h

列出所有选项并简要说明。

--version
-v

打印版本信息然后退出。

AWK

要使用的 Awk 解释器的完整文件名。如果未设置， PATH 将搜索一个 awk 程序。

GDB

一个 GDB 可执行文件的完整文件名，用于生成堆栈回溯信息。 如果未设置， PATH 将搜索一个 gdb 程序。

GDBARGS

要传递给 gdb 程序的可选参数。
