15.4.1.2 C 和 C++++ 常量 ¶

GDB 允许你用以下方式表示 C 和 C++ 的常量：++

  • 整数常量是一串数字。八进制常量以 ‘0’（即零）开头，十六进制常量以 ‘0x’ 或 ‘0X’ 开头。常量也可以以字母 ‘l’ 结尾，表示该常量应被视为一个 long 值。
  • 浮点常量是一串数字，后面跟着一个小数点，再跟着一串数字，还可以可选地跟着一个指数。指数的形式为： ‘e[[+]|-]nnn’，其中 nnn 是另一串数字。对于正指数，‘+’ 是可选的。 浮点常量也可以以字母 ‘f’ 或 ‘F’ 结尾，表示该常量应被视为 float （而不是默认的 double) 类型；或者以字母 ‘l’ 或 ‘L’ 结尾，表示一个 long double 常量。
  • 枚举常量由枚举标识符或它们的整数值组成。
  • 字符常量是用单引号括起来的单个字符（'），或者是一个数字——对应字符的序号值（通常是其 ASCII 值）。在引号内，单个字符可以用字母或 转义序列, 其形式为 ‘\nnn’，其中 nnn 是字符序号值的八进制表示；或者形式为 ‘\x’，其中 ‘x’ 是一个预定义的特殊字符——例如， ‘\n’ 表示换行符。

    宽字符常量可以像在 C 语言中一样，在字符常量前加 ‘L’ 来表示。例如，‘L'x'’ 是 ‘x’ 的宽字符形式。计算该常量的值时使用目标宽字符集（见 字符集)。

  • 字符串常量是用双引号括起来的一串字符常量（")。任何有效的字符常量（如上所述）都可以出现。字符串中的双引号必须用反斜杠转义，例如 ‘"a\"b'c"’ 是一个包含五个字符的字符串。

    宽字符串常量可以像在 C 语言中一样，在字符串常量前加 ‘L’ 来表示。计算该常量的值时使用目标宽字符集（见 字符集)。

  • 指针常量是一个整数值。你也可以使用 C 运算符 ‘&’ 来表示指向常量的指针。
  • 数组常量是用花括号 ‘{’ 和 ‘}’ 括起来的逗号分隔列表；例如，‘{1,2,3}’ 是一个包含三个整数元素的数组，‘{{1,2}, {3,4}, {5,6}}’ 是一个三行两列的数组， 而 ‘{&"你好", &"那里", &"弗雷德"}’ 是一个包含三个元素的指针数组。
