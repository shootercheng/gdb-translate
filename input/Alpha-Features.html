G.5.2 Alpha Features ¶

The ‘org.gnu.gdb.alpha.core’ feature is required for Alpha targets. It must contain the following 64-bit registers; note that GDB uses the software names for Alpha registers:

  • ‘v0’: function return value
  • ‘t0’ through ‘t12’: temporary registers
  • ‘s0’ through ‘s5’: saved registers
  • ‘fp’: frame pointer
  • ‘a0’ through ‘a5’: argument registers
  • ‘ra’: return address
  • ‘at’: assembler temporary register
  • ‘gp’: global pointer
  • ‘sp’: stack pointer
  • ‘zero’: always zero
  • ‘f0’ through ‘f30’: floating-point registers
  • ‘fpcr’: floating-point control register
  • ‘pc’: program counter
  • ‘’: an anonymous register for historical purpose
  • ‘unique’: PALcode memory slot
