23.3.2 Python API ¶

通过发出命令python help (gdb)，你可以获取GDB的Python API的快速在线帮助。

具有两个或更多可选参数的函数和方法允许使用关键字语法来指定它们。这允许传递一些可选参数，同时跳过其他参数。示例：gdb.some_function ('foo', bar = 1, baz = 2)。
