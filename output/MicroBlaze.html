21.3.5 MicroBlaze ¶

MicroBlaze是一种软核处理器，在各种赛灵思FPGA上得到支持，如Spartan或Virtex系列。带有这些处理器的开发板通常具有JTAG端口，可连接到运行赛灵思嵌入式开发套件（EDK）或软件开发套件（SDK）的主机系统。 这个主机系统用于将配置比特流下载到目标FPGA。赛灵思微处理器调试器（XMD）程序通过JTAG接口与目标开发板通信，并为开发板提供一个gdbserver接口。默认情况下，xmd使用端口1234。（虽然可以更改此默认端口，但需要使用未记录的xmd命令。如果需要这样做，请联系赛灵思技术支持。）

使用以下GDB命令连接到MicroBlaze目标处理器。

target remote :1234

如果您在与xmd相同的系统上运行GDB，请使用此命令连接到目标。

target remote xmd-host:1234

如果目标连接到在名为xmd-host的不同系统上运行的xmd，请使用此命令连接到目标。

load

使用此命令将程序下载到MicroBlaze目标。

set debug microblaze n

如果非零，则启用特定于MicroBlaze的调试消息。

show debug microblaze n

显示特定于MicroBlaze的调试级别。
