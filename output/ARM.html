21.3.2 ARM ¶

GDB提供了以下特定于ARM的命令：

set arm disassembler ¶

此命令从反汇编样式列表中进行选择。"std"样式是标准样式。

show arm disassembler ¶

显示当前的反汇编样式。

set arm apcs32 ¶

此命令在32位和26位之间切换ARM操作模式。

show arm apcs32

显示ARM 32位模式的当前使用情况。

set arm fpu fputype

此命令设置ARM浮点单元（FPU）类型。参数fputype可以是以下之一：

auto

通过查询操作系统ABI来确定FPU类型。

softfpa

软件FPU，在小端ARM处理器上具有混合字节序的双精度数。

fpa

GCC编译的FPA协处理器。

softvfp

具有纯字节序双精度数的软件FPU。

vfp

VFP协处理器。

show arm fpu

显示FPU的当前类型。

set arm abi

此命令强制GDB使用指定的ABI。

show arm abi

显示当前使用的ABI。

set arm fallback-mode (arm|thumb|auto)

GDB在可用时使用符号表来确定指令是ARM还是Thumb。此命令控制GDB在符号表不可用时的默认行为。默认值是‘auto’，这会导致GDB使用当前执行模式（来自CPSR寄存器中的T位）。

show arm fallback-mode

显示当前的备用指令模式。

set arm force-mode (arm|thumb|auto)

此命令覆盖使用符号表来确定指令是ARM还是Thumb。默认值是‘auto’，这会导致GDB使用符号表，然后是‘set arm fallback-mode’的设置。

show arm force-mode

显示当前的强制指令模式。

set arm unwind-secure-frames

此命令在具有安全扩展的Cortex-M上启用从非安全模式到安全模式的展开。 这在展开异常堆栈时可能会触发安全异常。 它默认启用。

show arm unwind-secure-frames

显示是否启用从非安全模式到安全模式的展开。

set debug arm

切换是否显示来自ARM目标支持子系统的特定于ARM的调试消息。

show debug arm

显示是否启用特定于ARM的调试消息。

target sim [simargs] …

GDB ARM模拟器接受以下可选参数。

--swi-support=type

告诉模拟器支持哪些SWI接口。参数type可以是以下值的逗号分隔列表。 默认值是all。

none
demon
angel
redboot
all
