C.4 为主机和目标指定名称 ¶

configure脚本中用于主机和目标的规范基于一种三部分命名方案，但也支持一些简短的预定义别名。完整的命名方案按以下模式编码三条信息： 

架构-供应商-操作系统

例如，可以使用别名sun4作为主机参数，或作为--target = 目标选项中目标的值。等效的完整名称是‘sparc - sun - sunos4’。

GDB附带的configure脚本没有提供任何查询工具来列出所有支持的主机和目标名称或别名。configure调用Bourne shell脚本config.sub将缩写映射为完整名称；如果愿意，你可以阅读该脚本，或者可以使用它来测试你对缩写的猜测—例如： 

% sh config.sub i386 - linux
i386 - pc - linux - gnu
% sh config.sub alpha - linux
alpha - unknown - linux - gnu
% sh config.sub hp9k700
hppa1.1 - hp - hpux
% sh config.sub sun4
sparc - sun - sunos4.1.1
% sh config.sub sun3
m68k - sun - sunos4.1.1
% sh config.sub i986v
无效配置 `i986v'：未识别机器 `i986v'

config.sub也分布在GDB源目录中（对于17.2版本，在gdb - 17.2中）。
