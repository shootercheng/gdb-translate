3.5命令选项 ¶

一些命令接受以破折号开头的选项。例如，print -pretty。与命令名类似，如果缩写明确，你可以将GDB选项缩写为选项名的前几个字母，并且你还可以使用TAB键让GDB补全选项中单词的其余部分（或者，如果有多种可能性，向你展示可用的替代项）。

一些命令将原始输入作为参数。例如，print命令处理GDB支持的任何语言中的任意表达式。对于这样的命令，由于原始输入可能以破折号开头，这可能会与选项或其任何缩写混淆，例如print -p（print -pretty的缩写或打印负数p？），如果你指定任何命令选项，那么你必须使用双破折号（--）分隔符来表示选项的结束。

一些选项被描述为接受一个可以是on或off的参数。这些被称为布尔选项。与布尔设置命令类似，on和off是典型值，但1、yes和enable中的任何一个也可以用作“真”值，0、no和disable中的任何一个也可以用作“假”值。你也可以省略“真”值，因为它默认隐含。

例如，这些是等效的： 

(gdb) print -object on -pretty off -element unlimited -- *myptr
(gdb) p -o -p 0 -e u -- *myptr

你可以通过在命令名后输入破折号并按下TAB键来发现某个命令接受的选项集。例如： 

(gdb) print -TABTAB
-address         -max-depth               -object          -static-members
-array           -memory-tag-violations   -pretty          -symbol
-array-indexes   -nibbles                 -raw-values      -union
-elements        -null-stop               -repeats         -vtbl

在某些情况下，补全将用关于选项期望何种类型参数的建议来引导你。例如： 

(gdb) print -elements TABTAB
NUMBER     unlimited

这里，选项期望一个数字（例如，100），而不是字面的NUMBER。这样的元语法参数总是以大写形式呈现。
