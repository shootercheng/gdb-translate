23.3.2.30 符号的Python表示 ¶

GDB 将每个变量、函数和类型表示为符号表中的一个条目。请参阅 检查符号表。 类似地，Python使用 gdb.Symbol 对象在 GDB 中表示这些符号。

gdb 模块中提供了以下与符号相关的函数：

函数： gdb.lookup_symbol (name [, block [, domain]]) ¶

此函数按名称搜索符号。搜索范围可以限制在可选的域和块参数中定义的参数。

name 是符号的名称。它必须是一个字符串。可选的 block 参数将搜索限制为在该 block 中可见的符号。block 参数必须是一个 gdb.Block 对象。如果省略，则使用当前帧的块。可选的 domain 参数将搜索限制为域类型。domain 参数必须是在 gdb 模块中定义的域常量，并在本章后面描述。

结果是一个由两个元素组成的元组。 第一个元素是一个 gdb.Symbol 对象，如果未找到符号则为 None。 如果找到符号，第二个元素如果符号是方法对象的字段（例如，C++ 中的 this）则为 True，否则为 False。 如果未找到符号，第二个元素为 False。

函数： gdb.lookup_global_symbol (name [, domain]) ¶

此函数按名称搜索全局符号。 搜索范围可以由域参数限制。

name 是符号的名称。它必须是一个字符串。 可选的 domain 参数将搜索限制为域类型。 domain 参数必须是在 gdb 模块中定义的域常量，并在本章后面描述。

结果是一个 gdb.Symbol 对象，如果未找到符号则为 None。

函数： gdb.lookup_static_symbol (name [, domain]) ¶

此函数按名称搜索具有静态链接的全局符号。 搜索范围可以由域参数限制。

name 是符号的名称。它必须是一个字符串。 可选的 domain 参数将搜索限制为域类型。 domain 参数必须是在 gdb 模块中定义的域常量，并在本章后面描述。

结果是一个 gdb.Symbol 对象，如果未找到符号则为 None。

请注意，此函数不会找到函数作用域的静态变量。要查找此类变量，请遍历函数的 gdb.Block 的变量，并检查 block.addr_class 是否为 gdb.SYMBOL_LOC_STATIC。

可能有多个具有相同名称的具有静态链接的全局符号。此函数只会返回它找到的第一个匹配符号。找到哪个符号取决于 GDB 当前停止的位置，因为 GDB 首先会在当前目标文件中搜索匹配符号，然后搜索所有其他目标文件。如果应用程序尚未运行，则 GDB 会按调试信息中出现的顺序搜索所有目标文件。

函数： gdb.lookup_static_symbols (name [, domain]) ¶

与 gdb.lookup_static_symbol 类似，此函数按名称搜索具有静态链接的全局符号，并可选地由域参数限制。但是，此函数返回找到的所有匹配符号的列表，而不仅仅是第一个。

name 是符号的名称。它必须是一个字符串。 可选酌 domain 参数将搜索限制为域类型。 domain 参数必须是在 gdb 模块中定义的域常量，并在本章后面描述。

结果是一个 gdb.Symbol 对象列表，如果未找到匹配符号则可能为空。

请注意，此函数不会找到函数作用域的静态变量。要查找此类变量，请遍历函数的 gdb.Block 的变量，并检查 block.addr_class 是否为 gdb.SYMBOL_LOC_STATIC。

一个 gdb.Symbol 对象具有以下属性：

变量： Symbol.type ¶

符号的类型，如果没有记录类型则为 None。 此属性表示为一个 gdb.Type 对象。 请参阅 Python中的类型。此属性不可写。

变量： Symbol.symtab ¶

符号所在的符号表。此属性表示为一个 gdb.Symtab 对象。请参阅 Python中的符号表表示。此属性不可写。

变量： Symbol.line ¶

符号在源代码中定义的行号。 这是一个整数。

变量： Symbol.name
