23.3.2.19 Python中的线程 ¶

Python脚本可以通过gdb.InferiorThread类的对象来访问由GDB控制的下级线程的信息，并对其进行操作。

在gdb模块中可以使用以下与线程相关的函数：

函数：gdb.selected_thread () ¶

此函数返回选定线程的线程对象。如果没有选定的线程，将返回None。

要获取下级的线程列表，请使用Inferior.threads()方法。请参阅 Python中的下级。

一个gdb.InferiorThread对象具有以下属性：

变量：InferiorThread.name ¶

线程的名称。如果用户使用thread name指定了名称，则返回该名称。否则，如果有操作系统提供的名称，则返回该名称。否则，返回None。

此属性可以赋值。新值必须是一个字符串对象，用于设置新名称，或者是None，用于删除任何用户指定的线程名称。

变量：InferiorThread.num ¶

由GDB分配的下级线程编号。

变量：InferiorThread.global_num ¶

由GDB分配的线程全局ID。例如，你可以使用此ID使Python断点特定于线程（请参阅 断点.线程属性）。

变量：InferiorThread.ptid ¶

由操作系统分配的线程ID。此属性是一个包含三个整数的元组。第一个是进程ID（PID）；第二个是轻量级进程ID（LWPID），第三个是线程ID（TID）。LWPID或TID可能为0，表示操作系统不使用该标识符。

变量：InferiorThread.ptid_string ¶

此只读属性包含一个表示InferiorThread.ptid的字符串。这是GDB在 info threads 输出的“Target Id”列中使用的字符串（请参阅 ‘info threads’）。

变量：InferiorThread.inferior ¶

此线程所属的下级。此属性表示为一个gdb.Inferior对象。此属性不可写。

变量：InferiorThread.details ¶

一个包含目标特定线程状态信息的字符串。此字符串的格式因目标而异。如果此线程没有其他状态信息，则此属性包含None。

例如，在GNU/Linux系统上，正在退出的线程将返回字符串“Exiting”。对于远程目标，如果目标支持，将通过“qThreadExtraInfo”远程数据包获取details字符串（请参阅 ‘qThreadExtraInfo’）。

GDB在 info threads 输出的“Target Id”列中显示details字符串（请参阅 ‘info threads’）。

一个gdb.InferiorThread对象具有以下方法：

函数：InferiorThread.is_valid () ¶

如果gdb.InferiorThread对象有效，则返回True，否则返回False。如果线程退出，或者线程所属的下级被删除，gdb.InferiorThread对象将变为无效。如果在调用方法时对象无效，所有其他gdb.InferiorThread方法将引发异常。

函数：InferiorThread.switch () ¶

这会将GDB当前选定的线程更改为由此对象表示的线程。

函数：InferiorThread.is_stopped () ¶

返回一个布尔值，指示线程是否已停止。

函数：InferiorThread.is_running () ¶

返回一个布尔值，指示线程是否正在运行。

函数：InferiorThread.is_exited () ¶

返回一个布尔值，指示线程是否已退出。

函数：InferiorThread.handle () ¶

返回线程对象的句柄，以Python bytes 对象表示。句柄的 gdb.Value 表示可以通过 gdb.Value(bufobj, type) 构造，其中 bufobj 是句柄的Python bytes 表示，type 是句柄类型的 gdb.Type。

可以使用常规的Python方式向gdb.InferiorThread对象添加任意属性。例如，如果需要记录与线程相关的一些额外信息，这将很有用。

有关为新属性选择合适名称的指导，请参阅 选择属性名称。

在这个人为编造的示例中，我们记录线程最后停止的时间： 

(gdb) python
import datetime

def thread_stopped(event):
    if event.inferior_thread is not None:
        thread = event.inferior_thread
    else:
        thread = gdb.selected_thread()
    thread._last_stop_time = datetime.datetime.today()

gdb.events.stop.connect(thread_stopped)
