22.8.1自动加载当前目录中的初始化文件 ¶

默认情况下，GDB会读取并执行当前工作目录中初始化文件（如果有）里预先设置好的命令序列， 请参阅启动期间当前目录中的初始化文件。

请注意，加载此本地的.gdbinit文件还需要相应配置好的auto-load safe-path（请参阅自动加载的安全限制）。

set auto-load local-gdbinit [on|off] ¶

启用或禁用在当前目录的初始化文件中查找预先设置好的命令序列（请参阅预先设置好的命令序列）的自动加载功能。

show auto-load local-gdbinit ¶

显示是否启用了从当前目录初始化文件中自动加载预先设置好的命令序列功能。

info auto-load local-gdbinit ¶

打印当前目录初始化文件中的预先设置好的命令序列是否已被自动加载。
