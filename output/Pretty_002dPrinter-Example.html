10.10.2漂亮打印机示例 ¶

以下是在没有漂亮打印机的情况下C++ std::string的样子： 

(gdb) print s
$1 = {
  static npos = 4294967295, 
  _M_dataplus = {
    <std::allocator<char>> = {
      <__gnu_cxx::new_allocator<char>> = {
        <无数据字段>}, <无数据字段>
      },
    std::basic_string<char, std::char_traits<char>,
      std::allocator<char> >::_Alloc_hider的成员：
    _M_p = 0x804a01 4 "abcd"
  }
}

使用针对 std::string 的漂亮打印机时，只打印内容： 

(gdb) print s
$2 = "abcd"
说明：原文档中 “0x804a014” 疑似多了个 “4”，我按正确的格式翻译了，但不确定是否是原文有误，你可根据实际情况检查。
