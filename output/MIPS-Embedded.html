21.3.6 MIPS嵌入式 ¶

GDB支持针对MIPS目标的这些特殊命令：

set mipsfpu double ¶
set mipsfpu single
set mipsfpu none
set mipsfpu auto
show mipsfpu

如果你的目标板不支持MIPS浮点协处理器，你应该使用命令‘set mipsfpu none’（如果你需要这样做，你可能希望将该命令放入你的GDB初始化文件中）。这会告诉GDB如何找到返回浮点值的函数的返回值。它还允许GDB在调用板上的函数时避免保存浮点寄存器。如果你正在使用仅支持单精度浮点的浮点协处理器，如在R4650处理器上，使用命令‘set mipsfpu single’。可以使用‘set mipsfpu double’选择默认的双精度浮点协处理器。

在以前的版本中，唯一的选择是双精度或无浮点，所以‘set mipsfpu on’将选择双精度，‘set mipsfpu off’将选择无浮点。

像往常一样，你可以使用‘show mipsfpu’查询‘mipsfpu’变量。
