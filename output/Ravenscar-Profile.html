15.4.10.9 使用Ravenscar配置文件时的任务支持 ¶

Ravenscar配置文件是Ada任务功能的一个子集，专门为有安全关键实时要求的系统设计。

set ravenscar task-switching on ¶

在调试使用Ravenscar配置文件的程序时允许任务切换。这是默认设置。

set ravenscar task-switching off ¶

在调试使用Ravenscar配置文件的程序时关闭任务切换。这主要是为了禁用添加对Ravenscar配置文件支持的代码，以防GDB或Ravenscar运行时中的错误导致GDB无法正常工作。要使其生效，请在程序启动前运行此命令。

show ravenscar task-switching ¶

显示在使用Ravenscar配置文件的程序中是否可以从一个任务切换到另一个任务。

当启用Ravenscar任务切换时，GDB会将Ravenscar任务宣布为好像它们是线程一样： 

(gdb) continue
[New Ravenscar Thread

Ravenscar任务和底层CPU线程都会出现在 info threads 的输出中： 

(gdb) info threads
  Id   Target Id                  Frame
  一个已知的限制是，目前无法单步执行运行时初始化序列。如果你需要调试这段代码，你应该使用 set ravenscar task-switching off。
