23.3.2.15 Xmethod API ¶

GDB Python API提供了用于实现、注册和操作x方法的类接口和函数。 请参阅 Python中的Xmethods。

一个x方法匹配器应该是从 gdb.xmethod 模块中定义的 XMethodMatcher 类派生的类的实例，或者是具有类似接口和属性的对象。 XMethodMatcher 的实例具有以下属性：

变量： name ¶

匹配器的名称。

变量： enabled ¶

一个布尔值，表示匹配器是启用还是禁用。

变量： methods ¶

由匹配器管理的命名方法列表。 列表中的每个对象都是 gdb.xmethod 模块中定义的 XMethod 类的实例，或者是具有以下属性的任何对象：

name

x方法的名称，对于由匹配器管理的每个x方法，该名称应该是唯一的。

enabled

一个布尔值，表示x方法是启用还是禁用。

XMethod 类是一个方便类具有与上述相同的属性以及以下构造函数：

函数： XMethod.__init__ (self, name) ¶

构造一个名称为 name 的启用的x方法。

XMethodMatcher 类具有以下方法：

函数： XMethodMatcher.__init__ (self, name) ¶
