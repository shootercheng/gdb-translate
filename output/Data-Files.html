18.8 GDB数据文件 ¶

GDB有时会读取辅助数据文件。这些文件保存在一个称为数据目录的目录中。

你可以设置数据目录的名称，并查看GDB当前正在使用的名称。

set data-directory directory ¶

将GDB搜索辅助数据文件的目录设置为directory。

show data-directory ¶

显示GDB搜索辅助数据文件的目录。

你可以通过使用配置时的‘--with-gdb-datadir’选项来设置默认数据目录。如果数据目录在GDB配置的二进制前缀（用‘--prefix’或‘--exec-prefix’设置）内，那么如果已安装的GDB被移动到新位置，默认数据目录将自动更新。

也可以使用--data-directory命令行选项指定数据目录。 请参阅 选择模式。
