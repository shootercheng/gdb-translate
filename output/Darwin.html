21.1.6达尔文 ¶

GDB提供了以下特定于达尔文目标的命令：

set debug darwin num ¶

当设置为非零值时，启用特定于达尔文支持的调试消息。值越高，输出越详细。

show debug darwin ¶

显示达尔文消息的当前状态。

set debug mach-o num ¶

当设置为非零值时，在GDB读取达尔文目标文件时启用调试消息。（Mach-O是达尔文上用于目标和可执行文件的文件格式。）值越高，输出越详细。这是一个用于诊断GDB内部问题的命令，正常使用时不需要。

show debug mach-o ¶

显示Mach-O文件消息的当前状态。

set mach-exceptions on ¶
set mach-exceptions off

在达尔文上，可以将故障首先报告为Mach异常，然后映射到Posix信号。使用此命令在下级进程中开启对Mach异常的捕获。这有时可能有助于更好地理解故障原因。默认是关闭的。

show mach-exceptions ¶

显示异常捕获的当前状态。
