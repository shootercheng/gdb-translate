33.3.1 Readline初始化文件语法 ¶

Readline初始化文件中只允许使用一些基本结构。空行将被忽略。 以“#”开头的行是注释。 以“$”开头的行表示条件结构（见条件初始化结构）。其他行表示变量设置和键绑定。

变量设置

你可以通过在初始化文件中使用set命令来改变Readline中的变量值，从而修改Readline的运行时行为。 语法很简单： 

set 变量 值

例如，以下是如何从默认的类似Emacs的键绑定改为使用vi行编辑命令： 

set editing-mode vi

变量名和值在适当的时候不区分大小写。未识别的变量名将被忽略。

布尔变量（那些可以设置为开或关的变量）如果值为空或空字符串、on（不区分大小写）或1，则设置为开。任何其他值都会导致变量被设置为关。
