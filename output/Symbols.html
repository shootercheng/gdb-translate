16 检查符号表 ¶

本章中描述的命令允许你查询程序中定义的符号（变量、函数和类型）。此信息是程序文本中固有的，并且不会随着程序的执行而改变。GDB在你启动GDB时指定的文件（见选择文件）或通过文件管理命令之一（见指定文件的命令）找到的程序符号表中查找此信息。

偶尔，你可能需要引用包含特殊字符的符号，而GDB通常将这些字符视为单词分隔符。最常见的情况是引用其他源文件中的静态变量（见程序变量）。文件名作为调试符号记录在目标文件中，但GDB通常会将典型的文件名（如foo.c）解析为三个单词：“foo”、“.”、“c”。为了让GDB将“foo.c”识别为单个符号，请将其用单引号括起来；例如， 

p 'foo.c'::x

在文件foo.c的作用域中查找变量x的值。

set case-sensitive on ¶
set case-sensitive off
set case-sensitive auto

通常，当GDB查找符号时，它会根据当前源语言确定的大小写敏感性来匹配符号名称。偶尔，你可能希望控制这一点。命令set case-sensitive允许你通过指定on进行大小写敏感匹配，或指定off进行大小写不敏感匹配。如果你指定auto，大小写敏感性将重置为适合源语言的默认值。除了Fortran语言默认大小写不敏感匹配外，所有语言的默认值都是大小写敏感匹配。

show case-sensitive ¶

此命令显示符号查找的当前大小写敏感性设置。

set print type methods ¶
set print type methods on
set print type methods off

通常，当GDB打印一个类时，它会显示该类中声明的任何方法。你可以通过将适当的标志传递给ptype，或使用命令set print type methods来控制此行为。指定on将导致GDB显示方法；这是默认设置。指定off将导致GDB省略方法。

show print type methods ¶

此命令显示打印类时方法显示的当前设置。

set print type nested-type-limit limit ¶
set print type nested-type-limit unlimited

设置类型打印机将显示的嵌套类型的显示限制。限制为“unlimited”或“-1”将显示所有嵌套定义。默认情况下，类型打印机不会显示类中定义的任何嵌套类型。

show print type nested-type-limit ¶

此命令显示打印类时嵌套类型的当前显示限制。

set print type typedefs ¶
set print type typedefs on
set print type typedefs off
