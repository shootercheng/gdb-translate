write ¶

概要：
int write(int fd, const void *buf, unsigned int count);
请求：

‘Fwrite,fd,bufptr,count’

返回值：

成功时，返回写入的字节数。 零表示没有写入任何内容。出错时，返回 -1。

错误：
EBADF

fd 不是有效的文件描述符，或者不是用于写入而打开的。

EFAULT

bufptr 是无效的指针值。

EFBIG

尝试写入超过主机特定允许的最大文件大小的文件。

ENOSPC

设备上没有空间写入数据。

EINTR

调用被用户中断。
