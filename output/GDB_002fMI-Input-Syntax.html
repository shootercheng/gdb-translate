27.2.1 GDB/MI输入语法 ¶

命令 →

cli命令 | mi命令

cli命令 →

[ 令牌 ] cli命令 换行符，其中 cli命令是任何现有的GDB CLI命令。

mi命令 →

[ 令牌 ] "-" 操作 ( " " 选项 )* [ " --" ] ( " " 参数 )* 换行符

令牌 →

"任意数字序列"

选项 →

"-" 参数 [ " " 参数 ]

参数 →

非空白序列 | c字符串

操作 →

本章中描述的任何操作

非空白序列 →

任何内容，只要它不包含特殊字符，如 "-"、换行符、"""，当然还有 " "

c字符串 →

""" 七位ISO c字符串内容 """

换行符 →

CR
| CR-LF

注意：

  • CLI命令仍由MI解释器处理；其 输出将在下面描述。
  • 令牌（如果存在）在命令 完成时会被传回。
  • 一些MI命令接受可选参数作为参数 列表的一部分。 每个选项由一个前导 ‘-’（破折号）标识，并且可以 后跟一个可选的参数参数。 选项在参数列表中首先出现，并且可以使用 ‘--’ 与普通参数分隔开（当一些参数以破折号开头时这很有用）。

实用方法：

  • 我们希望能够轻松访问现有的CLI语法（用于调试）。
  • 我们希望能够轻松识别MI操作。
