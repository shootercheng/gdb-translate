E.14.10 文件I/O示例 ¶

写调用的示例序列，文件描述符3，缓冲区位于目标地址0x1234，应写入6个字节： 

<- Fwrite,3,1234,6
请求从目标读取内存
-> m1234,6
<- XXXXXX
返回“已写入6个字节”
-> F6

读调用的示例序列，文件描述符3，缓冲区位于目标地址0x1234，应读取6个字节： 

<- Fread,3,1234,6
请求向目标写入内存
-> X1234,6:XXXXXX
返回“已读取6个字节”
-> F6

读调用的示例序列，由于文件描述符无效（EBADF），主机上的调用失败： 

<- Fread,3,1234,6
-> F-1,9

读调用的示例序列，在主机上调用系统调用之前用户按下Ctrl-c： 

<- Fread,3,1234,6
-> F-1,4,C
<- T02

读调用的示例序列，在主机上调用系统调用之后用户按下Ctrl-c： 

<- Fread,3,1234,6
-> X1234,6:XXXXXX
<- T02
