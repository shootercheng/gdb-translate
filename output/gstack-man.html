gstack ¶

gstack [-h | --help] [-v | --version] pid

打印具有进程ID pid 的正在运行程序的堆栈跟踪。如果进程是多线程的，gstack 会输出该进程中每个存在的线程的回溯。

该脚本调用 GDB，附加到给定的进程ID，打印堆栈跟踪，然后从该进程分离。

如果 gdb 由于任何原因无法附加到给定的进程ID，例如进程ID不存在或没有足够的权限附加到该进程，gstack 将以非零状态退出。

--help
-h

列出所有选项，并给出简要解释。

--version
-v

打印版本信息然后退出。

AWK

要使用的Awk解释器的完整文件名。如果未设置，将在 PATH 中搜索 awk 程序。

GDB

用于生成堆栈回溯的 GDB 可执行文件的完整文件名。如果未设置，将在 PATH 中搜索 gdb 程序。

GDBARGS

要传递给 gdb 程序的可选参数。
