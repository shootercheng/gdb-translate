15.4支持的语言 ¶

GDB支持C、C++、D、Go、Objective-C、Fortran、OpenCL C、Pascal、Rust、汇编、Modula-2和Ada。 无论使用何种语言，GDB的某些功能都可用于表达式：GDB的@和::运算符，以及“{type}addr”结构（见表达式）可与任何支持语言的结构一起使用。

以下各节详细介绍了GDB对每种源语言的支持程度。这些章节并非语言教程或参考手册，而仅作为GDB表达式解析器接受内容以及不同语言的输入和输出格式应为何样的参考指南。关于这些语言中的每一种都有许多优秀的书籍；请查阅这些书籍以获取语言参考或教程。

  • C和C++
  • D
  • Go
  • Objective-C
  • OpenCL C
  • Fortran
  • Pascal
  • Rust
  • Modula-2
  • Ada
