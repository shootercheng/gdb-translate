21.3.10 CRIS ¶

当配置为调试CRIS时，GDB提供以下特定于CRIS的命令：

set cris-version ver ¶

将当前CRIS版本设置为ver，可以是‘10’或‘32’。 CRIS版本会影响寄存器名称和大小。当CRIS版本自动检测失败时，此命令很有用。

show cris-version

显示当前CRIS版本。

set cris-dwarf2-cfi ¶

设置用于CRIS调试的DWARF - 2 CFI的使用情况。默认值是‘on’。 当使用版本低于R59的gcc - cris时，更改为‘off’。

show cris-dwarf2-cfi

显示使用DWARF - 2 CFI的当前状态。

set cris-mode mode ¶

将当前CRIS模式设置为mode。仅在以guru模式调试时才应更改，在这种情况下应将其设置为‘guru’（默认值是‘normal’）。

show cris-mode

显示当前CRIS模式。
