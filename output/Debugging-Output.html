22.10关于内部事件的可选消息 ¶

GDB有一些命令可以启用来自各种GDB子系统的可选调试消息；通常这些命令对GDB维护人员或在报告错误时很有用。本节记录这些命令。

set exec-done-display ¶

打开或关闭异步命令完成的通知。开启后，当异步命令完成执行时，GDB将打印一条消息。默认是关闭。

show exec-done-display

显示异步命令完成通知的当前设置。

set debug aarch64 ¶

打开或关闭与ARM AArch64相关的调试消息显示。默认是关闭。

show debug aarch64

显示与ARM AArch64相关的调试消息显示的当前状态。

set debug arch ¶

打开或关闭gdbarch调试信息的显示。默认是关闭

show debug arch

显示gdbarch调试信息显示的当前状态。

set debug aix-thread ¶

显示有关AIX线程模块内部工作的调试消息。

show debug aix-thread
