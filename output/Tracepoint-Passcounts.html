13.1.3跟踪点通过计数 ¶

passcount [n [num]] ¶

设置跟踪点的通过计数。通过计数是一种自动停止跟踪实验的方法。如果一个跟踪点的通过计数是n，那么在该跟踪点第n次被命中时，跟踪实验将自动停止。如果未指定跟踪点编号num，则passcount命令将设置最近定义的跟踪点的通过计数。如果未给出通过计数，跟踪实验将一直运行，直到用户明确停止。

示例： 

(gdb) passcount 5 2 // 在跟踪点2的第5次执行时停止

                                   // 


(gdb) passcount 12  // 在最近定义的跟踪点的第12次执行时停止

                                   // 

(gdb) trace foo
(gdb) pass 3
(gdb) trace bar
(gdb) pass 2
(gdb) trace baz
(gdb) pass 1        // 当foo被执行3次或者bar被执行2次

                                    // 或者baz被执行1次时停止跟踪

                                    //
