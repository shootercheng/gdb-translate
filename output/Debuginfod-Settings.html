K .1 Debuginfod设置 ¶

GDB提供了以下用于配置debuginfod的命令。

set debuginfod enabled ¶
set debuginfod enabled on

GDB可以向debuginfod服务器查询缺失的调试信息和源文件。GDB还可以下载单个ELF/DWARF段，如.gdb_index，以帮助减少从debuginfod服务器下载的数据总量；这可以通过maint set debuginfod download-sections来控制（见 maint set debuginfod download-sections）。

set debuginfod enabled off

当缺少调试信息或源文件时，GDB不会尝试向debuginfod服务器查询。默认情况下，对于非交互式会话，debuginfod enabled设置为off。

set debuginfod enabled ask

GDB会在尝试执行下一次查询之前提示用户启用或禁用debuginfod。默认情况下，对于交互式会话debuginfod enabled设置为ask。

show debuginfod enabled ¶

显示debuginfod enabled是否设置为on、off或ask。

set debuginfod urls ¶
set debuginfod urls urls

设置debuginfod将尝试查询的以空格分隔的URL列表。仅应使用http://、https://和file://协议。debuginfod urls的默认值是从DEBUGINFOD_URLS环境变量复制而来的。

show debuginfod urls ¶

显示debuginfod将尝试查询的URL列表。

set debuginfod verbose ¶
set debuginfod verbose n

启用或禁用与debuginfod相关的输出。使用非零值启用，0禁用。默认情况下显示debuginfod输出。

show debuginfod verbose ¶

显示当前的详细程度设置。
