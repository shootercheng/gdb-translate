10.13 便利函数 ¶

GDB 还提供了一些 便利函数。这些函数的语法与便利变量类似。便利函数可以像普通函数一样在表达式中使用；然而，便利函数是在 GDB 内部实现的。

这些函数不需要 GDB 配置Python支持，这意味着它们始终可用。

$_isvoid (expr) ¶

如果表达式 expr 是 void，则返回1。否则返回0。

一个 void 表达式是指结果类型为 void 的表达式。例如，你可以检查一个便利变量（见 便利变量），以检查它是否是 void： 

(gdb) print $_exitcode
$1 = void
(gdb) print $_isvoid ($_exitcode)
$2 = 1
(gdb) run
启动程序：./a.out
[进程1（进程29572）正常退出]
(gdb) print $_exitcode
$3 = 0    
(gdb) print $_isvoid ($_exitcode)
$4 = 0

在上面的例子中，我们在调试的程序执行之前和之后使用 $_isvoid 来检查 $_exitcode 是否是 void。在执行之前，没有退出码可供检查，因此 $_exitcode 是 void。在执行之后调试的程序返回零，因此 $_exitcode 是零，这意味着它不再是 void。

void 表达式也可以是对正在调试的程序中的函数的调用。例如，给定以下函数： 

void
foo (void)
{
}

在 GDB 中调用它的结果是 void： 

(gdb) print foo ()
$1 = void
(gdb) print $_isvoid (foo ())
$2 = 1
(gdb) set $v = foo ()
(gdb) print $v
$3 = void
(gdb) print $_isvoid ($v)
$4 = 1
$_gdb_setting_str (setting) ¶

将 GDB setting 的值作为字符串返回。setting 是任何可以在 set 或 show 命令中使用的设置（见 控制 GDB）。 

(gdb) show print frame-arguments
打印非标量帧参数为 "scalars"。
(gdb) p $_gdb_setting_str("print frame-arguments")
$1 = "scalars"
(gdb) p $_gdb_setting_str("height")
$2 = "30"
(gdb)
$_gdb_setting (setting) ¶

返回 GDB setting 的值。 返回值的类型取决于设置。

布尔和自动布尔设置的值类型是 int。布尔值 off 和 on 分别转换为整数值 0 和 1。值 auto 转换为值 -1。

整数设置的值类型是 unsigned int 或 int，具体取决于设置。

一些整数设置接受 unlimited 值。根据设置，set 命令也接受值 0 或值 -1 作为 unlimited 的同义词。 例如，set height unlimited 等同于 set height 0。

一些其他接受 unlimited 值的设置使用值 0 表示零。 例如 set history size 0 表示不记录任何 GDB 命令历史。 对于这样的设置，-1 是 unlimited 的同义词。

有关与 unlimited 等效的数值，请参阅相应 set 命令的文档。

$_gdb_setting 函数根据 set 命令的使用方式将无限值转换为 0 或 -1 值。 

(gdb) p $_gdb_setting_str("height")
$1 = "30"
(gdb) p $_gdb_setting("height")
$2 = 30
(gdb) set height unlimited
(gdb) p $_gdb_setting_str("height")
$3 = "unlimited"
(gdb) p $_gdb_setting("height")
$4 = 0

(gdb) p $_gdb_setting_str("history size")
$5 = "unlimited"
(gdb) p $_gdb_setting("history size")
$6 = -1
(gdb) p $_gdb_setting_str("disassemble-next-line")
$7 = "auto"
(gdb) p $_gdb_setting("disassemble-next-line")
$8 = -1
(gdb)

其他设置类型（枚举、文件名、可选文件名、字符串、无转义字符串）作为字符串值返回。
