25.6 TUI配置变量 ¶

有几个配置变量控制TUI窗口的外观。

set tui border-kind 种类 ¶

为源窗口、汇编窗口和寄存器窗口选择边框外观。 可能的值如下：

space

使用空格字符绘制边框。

ascii

使用ASCII字符‘+’、‘-’和‘|’绘制边框。

acs

使用备用字符集绘制边框。如果终端支持，边框将使用字符线图形绘制。

set tui border-mode 模式 ¶
set tui active-border-mode 模式

为非活动窗口或活动窗口的边框选择显示属性。模式可以是以下之一：

normal

使用正常属性显示边框。

standout

使用突出显示模式。

reverse

使用反显模式。

half

使用半亮模式。

half-standout

使用半亮和突出显示模式。

bold

使用超亮或粗体模式。

bold-standout

使用超亮或粗体和突出显示模式。

set tui tab-width 字符数 ¶

将制表符停止位的宽度设置为字符数个字符。此设置会影响源窗口和汇编窗口中制表符的显示。

set tui compact-source [on|off] ¶

设置TUI源窗口是否以“紧凑”形式显示。默认显示为行号使用更多空间；紧凑显示仅使用当前文件中行号所需的空间。

set tui mouse-events [on|off] ¶

开启时（默认），鼠标点击控制TUI（见 TUI鼠标支持）。 关闭时，鼠标点击由终端处理，可以进行终端原生的文本选择。

set debug tui [on|off] ¶

开启或关闭与TUI相关的GDB内部调试消息的显示。

show debug tui ¶

显示与TUI相关的当前GDB内部调试消息的显示状态。

请注意，可以使用适当 的set style命令控制TUI边框的颜色。 见 输出样式。
