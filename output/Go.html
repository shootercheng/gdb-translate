15.4.3 Go ¶

GDB可用于调试用Go编写并使用gccgo或6g编译器编译的程序。

以下是特定于Go的功能和限制的总结：

当前的Go包 ¶

指定全局变量和函数时无需指定当前包的名称。

例如，给定程序： 

package main
var myglob = "Shall we?"
func main () {
  // ...
}

在main内部停止时，以下任何一个都可行： 

(gdb)p myglob
(gdb)p main.myglob
Go内置类型 ¶

GDB可识别string类型，并将其作为字符串打印。

Go内置函数 ¶

GDB表达式解析器可识别unsafe.Sizeof函数并在内部进行处理。

Go表达式的限制 ¶

除了&^之外，所有Go运算符均受支持。 Go的_“空白标识符”不受支持。 不支持指针的自动解引用。
