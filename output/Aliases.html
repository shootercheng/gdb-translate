23.2命令别名 ¶

别名允许你为现有命令定义替代拼写。 例如，如果在Python中定义的新GDB命令（见 使用Python扩展GDB）有一个很长的名字，那么有一个缩写版本会很方便，可以减少输入量。

GDB本身就使用别名。例如，‘s’是‘step’命令的别名，尽管它在其他情况下是‘set’和‘show’等其他命令的模糊缩写。

别名还用于提供多词命令的缩短或更常用版本。例如，GDB提供了‘tty’作为‘set inferior-tty’命令的别名。

你可以使用‘alias’命令定义新的别名。

alias [-a] [--] 别名 = 命令 [默认参数] ¶

别名指定新别名的名称。别名的每个单词必须由字母、数字、破折号和下划线组成。

命令指定要被别名化的现有命令的名称。

命令也可以是现有别名的名称。在这种情况下，命令不能是有默认参数的别名。

‘-a’选项指定新别名是命令的缩写。缩写在命令补全中不使用。

‘--’选项指定选项的结束，当别名以破折号开头时很有用。

你可以为你的别名指定 默认参数。这些默认参数将自动添加到在命令行上明确输入的别名参数之前。

例如，下面定义了一个别名 btfullall，它显示所有局部变量和所有帧参数： 

(gdb) alias btfullall = backtrace -full -frame-arguments all

有关默认参数的更多信息，请参阅 默认参数。

这里有一个简单的例子，展示了如何创建命令的缩写，以便减少输入量。假设你厌倦了输入‘disas’，它是‘disassemble’命令当前最短的明确缩写，而你想要一个更短的版本，名为‘di’。以下操作将实现这一点。 

(gdb) alias -a di = disas

请注意，别名与用户定义的命令不同。对于用户定义的命令，你还需要使用‘document’命令为其编写文档。别名会自动采用现有命令的文档。

这里有一个例子，我们将‘elms’作为‘set print elements’命令中‘elements’的缩写。这是为了展示你可以对命令的任何部分进行缩写。 

(gdb) alias -a set print elms = set print elements
(gdb) alias -a show print elms = show print elements
(gdb) set p elms 200
(gdb) show p elms
Limit on string chars or array elements to print is 200.

请注意，如果要定义‘set’命令的别名，并且想要为相应的‘show’命令定义别名，则需要分别定义后者。

在 命令 和 别名 中允许使用明确缩写的命令，就像通常一样。 

(gdb) alias -a set pr elms = set p ele

最后，这里有一个例子，展示了为更复杂的命令创建单字别名。 这创建了‘set print elements’命令的别名‘spe&rsquo。 

(gdb) alias spe = set print elements
(gdb) spe 20
